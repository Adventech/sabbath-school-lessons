---
title:  ദുഷ്ടശക്തികള്‍ക്കെതിരായ പോരാട്ടം
date:   14/09/2023
---

`എഫെ. 1:21; 3:10; 6:10-20 എന്നീ വാക്യങ്ങളില്‍ ചിത്രീകരിച്ചിരിക്കുന്ന ദുഷ്ട ആത്മീയ ശക്തികള്‍ക്കായി വിവിധ പേരുകള്‍ പട്ടികപ്പെടുത്തുന്നതില്‍ പൗലൊസിന്‍റെ ഉദ്ദേശ്യം എന്താണെന്ന് നിങ്ങള്‍ വിലയിരുത്തുന്നു?`

ഗുസ്തിക്കാര്‍ തമ്മിലുള്ള മത്സരത്തിനുള്ള ഒരു ഗ്രീക്കുപദം ഉപയോഗിച്ചുകൊണ്ട് പൗലൊസ് "നമ്മുടെ പോരാട്ടം" (എഫെ. 6:12) വിവരിക്കുന്നു (ുമഹല). യുദ്ധത്തിനായുള്ള ഏറ്റവും നല്ല തയ്യാറെടുപ്പായി മല്പിടിത്തത്തെ പരിഗണിച്ചിരുന്നതുകൊണ്ട്, സൈന്യങ്ങള്‍ ഏറ്റുമുട്ടുമ്പോള്‍ നടക്കുന്ന ആയുധത്തിനെതിരായ ആയുധത്തിന്‍റെയും കയ്യേറ്റത്തിന്‍റെയും ഉചിതമായ വിവരണമാണിത്. ദുഷ്ടശക്തികള്‍ക്കെതിരായ വിശ്വാസികളുടെ അടുത്ത പോരാട്ടത്തിന്‍റെ യാഥാര്‍ത്ഥ്യമാണ് പൗലൊസ് ഊന്നിപ്പറയുന്നത്. താന്‍ അവയ്ക്കു കൊടുക്കുന്ന പേരുകള്‍ ഇവയാണ്:

| എഫെസ്യര്‍ 1:21 | എഫെസ്യര്‍ 3:10 | എഫെസ്യര്‍ 6:12 |
| --- | --- | --- |
| എല്ലാ ഭരണാധികാരികളും (അല്ലെങ്കില്‍ എല്ലാ ഭരണവും) | ഭരണാധികാരികള്‍ | ഭരണാധികാരികള്‍ |
| (എല്ലാ) അധികാരവും | അധികാരങ്ങള്‍ | അധികാരങ്ങള്‍ |
| (എല്ലാ) ശക്തിയും | | ഈ അന്ധകാരത്തിന്മേലുള്ള പ്രാപഞ്ചിക ശക്തികള്‍ |
| (എല്ലാ) ആധിപത്യവും | | സ്വര്‍ല്ലോകങ്ങളിലെ ദുഷ്ടാത്മ സേനകള്‍ |
| വിളിക്കപ്പെടുന്ന സകല നാമവും | | |

പൗലൊസിന്‍റെ വിശാലമായ വിവരണങ്ങളില്‍ ("വിളിക്കപ്പെടുന്ന സകലനാമവും" എഫെ. 1:21, "സ്വര്‍ല്ലോകങ്ങളിലെ ദുഷ്ടാമശക്തികള്‍" എഫെ. 6:12) എല്ലാ തിന്മകളും അമാനുഷിക ശക്തികളും ക്രിസ്തുവിനു കീഴ്പെട്ടിരിക്കുന്നതായി പൗലൊസ് സ്ഥിരീകരിക്കുന്നു (എഫെ. 1:21). എന്നിരുന്നാലും ഏതു യുദ്ധത്തിലും എതിര്‍പക്ഷത്തെ ശക്തികളെ തരംതാഴ്ത്തി കാണുന്നത് ഒരിക്കലും നല്ല തന്ത്രമല്ല. നാം മനുഷ്യശത്രുക്കളോടല്ല പോരാടുന്നത്, എന്നാല്‍ സാത്താന്‍ എന്ന തന്ത്രശാലിയായ ഒരു പടത്തലവന്‍റെ നേതൃത്വത്തിലുള്ളസ്വര്‍ലോകങ്ങളിലെ ദുഷ്ടാത്മസേനയെ നമുക്കു നേരിടേണ്ടി വരുമെന്ന് പൗലൊസ് മുന്നറിയിപ്പു തരുന്നു (എഫെ. 6:11,12). എന്നിരുന്നാലും ശക്തരായ നമ്മുടെ ശത്രുക്കള്‍ക്കെതിരെ ജാഗ്രത പുലര്‍ത്തേണ്ട സമയത്ത് നാം അവരെ ഭയക്കേണ്ടതില്ല. യുദ്ധത്തില്‍ ദൈവം നമ്മോടൊപ്പമുണ്ട് (എഫെ. 6:10). കൂടാതെ അവന്‍ നമുക്ക് ഏറ്റവും നല്ല ആയുധം അവന്‍റെ സ്വന്തം കവചം "ദൈവത്തിന്‍റെ സര്‍വ്വായുധ വര്‍ഗ്ഗം" തന്നിരിക്കുന്നു (എഫെ. 6:11; യെശ. 59:15-17 താരതമ്യം ചെയ്യുക). അവന്‍റെ സത്യം, നീതി, സമാധാനം, വിശ്വാസം, രക്ഷ, പരിശുദ്ധാത്മാവ് എന്നിവ അവന്‍ നമ്മുടെ പക്കലാക്കിയിരിക്കുന്നു (എഫെ. 6:3-17). ദൈവം നമുക്കുമുമ്പില്‍ പോകുന്നതുകൊണ്ടും അവന്‍ നല്കിയ പടച്ചട്ടയില്‍ നാം തല മുതല്‍ കാല്‍ വരെ സജ്ജരാക്കിയിരിക്കകൊണ്ടും നമുക്കു പരാജയപ്പെടുവാന്‍ കഴിയുകയില്ല (റോമര്‍ 16:20; 1 കൊരി. 15:23,24; 2 തെസ്സ. 2:8).

`നാം തന്നെ തീര്‍ത്തും നിസ്സഹായരായിരിക്കുന്ന ഈ പ്രകൃത്യാതീത ദുഷ്ടശക്തികളുടെ യാഥാര്‍ത്ഥ്യം ഈ ശക്തികളെക്കാള്‍ വലിയവന്‍ എന്നുമാത്രമല്ല അവയെ തോല്പിച്ചു കഴിഞ്ഞ കര്‍ത്താവായ യേശുവില്‍ നാം എന്തുകൊണ്ട് മുറുകെ പിടിക്കണം എന്നതിനെക്കുറിച്ച് എന്താണ് നമ്മെ പഠിപ്പിക്കേണ്ടത്?`
