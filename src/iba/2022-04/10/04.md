---
title:  Orang Kudus Dalam Api Penuchi (Purgatory)
date:  29/11/2022
---

Ajar ke dikemeranka Gerija Roman Catholic madahka orang ke mati ke enda patut dibuai ngagai neraka tang bedau chukup besedia dibai ke serega tau dianjung dulu ngagai tempat dosa sida ulih dituchi lalu udah nya baru sida tau niki ngagai serega ari tempat nya. Pemerinsa enggau pemedis sida dalam api penuchi nya ulih dikurangka ngena sampi enggau duit dibayar kaban belayan kena nebus dosa sida nya.

Bup The Catechism of the Catholic Church silik amat nerangka pasal “purgatory” tauka api penuchi tu: “Semua orang ke mati dalam pengasih enggau kaul enggau Allah Taala, tang bedau chukup beresi dituchi, endang udah dijamin bulih pengidup ti meruan; tang udah sida ke mati nya sida deka dituchi dulu, ngambika sida tau bulih penuchi tauka pengudus ti ngasuh sida layak masuk ngagai pengandal serega.”—Catechism of the Catholic Church, (New York: Doubleday, 1995) p. 291. Iya mega madahka pemerinsa sida ba tempat nya ulih dikurangka ngena sampi kaban belayan sereta kereja badas sida ngarika sida ke udah mati nya. “Gerija mega ngansak sida meri sadekah (almsgiving), meli pengampun (indulgences), enggau kereja badas (penances) kena nebus dosa ti dikereja orang ngarika sida ke udah mati nya.”—Catechism of the Catholic Church, p. 291.

`Bacha Pengajar 9:10, Esekiel 18:20–22, enggau Hebrews 9:27. Baka ni ku ayat-ayat tu nyakal teori pasal ‘purgatory’ tauka api penuchi nya?`

Ajar senentang ‘purgatory’ (api penuchi) tu nyampurka ajar orang kapir pasal api neraka ti idup belama iya enggau pengawa orang kapir ti besampika sida ke udah mati. Ajar tu enda ulih diterima orang ti arapka ajar Bup Kudus pasal (1) orang ke mati nya diau belelak enda nemu diri dalam kubur sida (Pengajar 9:10); (2) pengelurus siku orang ke bedosa enda ulih dipindahka ngagai siku orang bedosa ke bukai (Esek. 18:20–22); (3) semina siku aja Seruan tauka Pengentara kitai, iya nya Jesus Kristus (1 Tim. 2:5); lalu (4) pemati ditangkanka pengakim tauka pechara ti penudi, enggau nadai bisi peluang ti kedua endur ngesal sereta berubah ari dosa dalam pengidup tu (Heb. 9:27).

Lalu implikasi ke lebih berat agi iya nya teori pasal purgatory ti ngelaban ajar Bup Kudus tu, udah ngemilutka perangai/karakter Allah Taala Empu. Iya ke bendar, “Pengawa Sitan kenyau ari pengelabuh iya endang deka meri gambar ke enda betul senentang Allah Taala. Iya ke udah meri penemu senentang ajar pasal semengat ke enda nemu mati nya. . . . Penemu pasal api neraka ke idup enda nemu padam nya mega datai ari Sitan; purgatory (api penuchi) tu mega ditempa Sitan. Semua ajar nya ngenyaika perangai Allah Taala, ngambika Iya dipeda baka ti kelalu, bebalaska dendam, enda lurus atur adat, sereta enggai ngampun penyalah.”—Ellen G. White, Manuscript 51 (Dec. 10), 1890. Ajar tu ukai madahka orang ke mati nya tinduk aja, nganti penatai Jesus baru, tang madahka sida nya diatu diengkah dalam api penuchi (purgatory), natka pemerinsa dipakai api dia, nyentukka bisi kaban belayan ulih nebus sida pansut ari nya.

`Nama ti ulih dipelajarka kitai ari ajar ti salah senentang api penuchi enggau api ti ngemerinsaka orang belama iya nya, pasal pemesai guna bisi doktrin tauka ajar ti betul? Nama kebuah utai ti dikarapka kitai nya penting, lalu ukai semina dalam sapa kitai arap?`
