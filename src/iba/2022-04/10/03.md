---
title:  Api Neraka
date:  28/11/2022
---

Dalam siti bup mit ti ditulis iya ungkup anak mit, ti bejuluk, The Sight of Hell (Dublin: James Duffy, [1874], p. 24), siku paderi Roman Catholic bebansa English, benama John Furniss (1809–1865) nyempaka orang ke diukum belama iya dalam neraka nya ngena siti bol ke besai, ke lebih besai ari langit enggau dunya. “Siku burung datai sekali dalam seratus juta taun lalu negu bol besi ke besai nya ngena siti bulu sayap iya.” Lalu ku penulis nya, orang ke bedosa deka terus ditunu dalam api neraka taja pen bol besi nya nyau abis diri ketegal ditegu bulu burung ke enda semaya datai nya!

Nyadi utai ti ngenyinuka kitai, iya nya, mayuh amat orang Protestant kemaya hari tu ke arapka ajar ti baka nya.

`Bacha Malaki 4:1 enggau Jude 7. Baka ni ku ayat-ayat tu ulih nyaup kitai mereti lebih manah agi pasal “api ti kekal” tauka, munyi ti disebut Jesus, orang ke bedosa deka “dibuai ngagai api ti enda nemu padam” (Matt. 18:8) (Mark 9:43, NKJV).`

Leka jaku “eternal” (kekal) (Heb. ‘olam; Greek aion, aionios) bisi beberapa bengkah reti, nitihka konteks iya dikena. Ngambika chunto, enti dikena bejaku pasal Allah Taala (Ul. Adat 33:27, ”everlasting”/belama iya), iya madahka Tuhan nya idup belama iya. Lebuh dikena bejaku pasal mensia (Exod. 21:6, “forever”/belama iya), iya semina ngemanjai umur mensia nya idup. Lebuh dikena bejaku pasal api (Matt. 18:8, Matt. 25:41, “everlasting”/kekal), iya madahka api nya enda deka padam nyentuk ngagai utai ti dipakai iya nya udah abis lengis angus. Reti nya, “api ti kekal” (“eternal fire”) nya dikumbai kekal laban iya nganguska orang ke jai sampai abis lengis, nyentukka sida “nadai agi urat tauka dan” (Mal. 4:1, NKJV).

Teori senentang ukum ti kekal ungkup orang ti jai tu nimbulka mayuh implikasi ti serius. Enti orang ti jai nya diukum belama iya, dia deh penyai enda deka ulih diapuska belama iya. Pia mega, semua pengidup mensia endang datai ari Allah Taala (Ul. Adat 32:39, Mas. 36:9), iya ke “enda rindu meda orang ke bedosa mati” (Esek. 33:11, NKJV). Enti baka nya, nama kebuah Iya majak meri pengidup ngagai orang ke jai ngambika sida diperinsa belama iya, deh? Kati, enda ka lebih betul agi enti Iya tembu ngapuska sida ngambika  nadai agi? Enti orang ke jai nya deka diakim lalu diukum “nitihka utai ti udah dikereja sida” (P’dang. 20:12, NKJV), nama kebuah pengidup mensia ke pandak aja nya diukum enda badu-badu belama-lama iya deh?

Semua ayat ke nyebut “eternal fire/api ti kekal” nya patut dianggap bejaku senentang “danau api” dalam Pemandang 20 (peda Pelajar 13), ti deka nyadi sapengudah timpuh seribu taun abis. Nya alai, enti mensia madahka api neraka nya udah bisi diatu lalu terus nunu belama iya, nya penemu ti nadai disukung Bup Kudus.

`Taja pen ajar ke enda betul pasal api neraka nya amat ngeruga kitai, nama ke dipadahka ajar ke betul senentang api neraka nya ngagai kitai senentang pengerindu Allah Taala, lebih agi enti dibandingka enggau ajar pasal ukum diperinsa belama iya dalam api neraka nya?`
