---
title:  「大巴比倫」
date:   06/06/2018
---

`請讀以下經文。它們向我們怎樣描述巴比倫呢？耶51：6，7，53，57；撒2：7；啟17：5，6；18：2，3。`

正如我們昨天所學的，巴比倫作為假敬拜的中心有著悠久的歷史；所以它是一個欺騙列國的末期權勢的貼切象徵。

`請比較龍，海獸和朱紅色的獸（啟12：3，13：1－3，17：3）。它們有什麼相似之處和不同之處呢？`

這三個獸都有七個頭和十隻角，這代表著但以理書第7章中四大獸的頭和角的總和。每個接續的帝國都是建立在先前的帝國之上。同樣，朱紅色的獸融合了龍和海獸的特點（各自象徵異教和教皇權的羅馬），以及陸獸的特點（啟13：11－14），「把作為上帝全部敵人的三種勢力結合成一個真實的聯盟。」──雅克•杜康，啟示錄的奧秘，希伯來人看啟示錄（黑格斯頓，馬里蘭州：回顧與展望出版協會，2002），原文第162頁。在啟示錄第17章，另有一特點是騎在朱紅色獸上的女人，朱紅色的首象徵著宗教和政治勢力的非法聯合。這個女人與啟示錄第12章中貞潔的婦人形成了顯明的對比。
 
| 貞潔的婦人（啟12）| 淫婦（啟17）|
| ------------ | ------------ |
| 在天上 | 在眾水之上 |
| 身披日頭 | 以金子、寶石、珍珠為妝飾 | 
| 頭戴十二星的冠冕 | 身穿朱紅色的衣服 |
| 受到龍的攻擊 | 受到龍的支持 | 
| 餘民之母 | 淫婦之母 |

作為「淫婦之母」，巴比倫一直不斷地在重蹈覆轍。背道的教會之母有很多背道的女兒。可是上帝不接受背道的基督教錯誤的教導和犯下的暴行。祂真實的百姓雖然受到撒旦的攻擊，但經過幾世紀後依然倖存。

啟14：8節已向人們警告過巴比倫的傾倒和背離真道。這一背道最終將引向由獸的印記發起的最後的欺騙（啟14：9－11）。這個警告將以更大的力量一再重複，它的終點將以最後一次呼喚仍在巴比倫的上帝百姓從中出來，並與上帝末期的教會聯合為結束（啟18：1－4）。
