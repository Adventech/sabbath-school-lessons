### Preberi pred proučevanjem
Jer 2,13; Ezk 36,25-27; Zah 14,8; Jn 3,26-30; Jn 4,1-42

> <p>Ključno besedilo</p>
> Ženi pa so govorili: »Ne verjamemo več zaradi tvojega pripovedovanja, kajti sami smo slišali in vemo, da je on resnično odrešenik sveta.« Jn 4,42

Kdo so bili Samarijani? Leta 722 pr. n. št. so severno izraelsko kraljestvo okupirali Asirci. Da bi vzpostavili politično stabilnost, so ujetnike razselili po vsem svojem kraljestvu. Tako so v severno izraelsko kraljestvo naselili ujetnike drugih narodov in ti ljudje so postali Samarijani, ki so prakticirali svojo obliko judovske vere.

Odnosi med njimi in Judi niso bili dobri. Na primer: Samarijani so zavirali obnovo templja, ko so se Judje vrnili iz Babilona, ker so medtem zgradili svoj tempelj na gori Gerizim, ki ga je leta 128 pr. n. št. uničil judovski vladar Janez Hirkan.

V Kristusovem času se je to sovraštvo nadaljevalo. Judje so se izogibali Samariji, kolikor se je le dalo. Čeprav so trgovali med seboj, so bili drugi stiki prepovedani. Judje si niso niti izposojali od Samarijanov niti niso hoteli sprejeti njihove pomoči. V tem kontekstu apostol Janez pripoveduje o srečanju med Jezusom, žensko ob vodnjaku in Samarijani, ki so prebivali v mestu Sihár.
