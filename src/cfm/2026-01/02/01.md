### Tuizarh zirlai hrang siar ding
Fil. 1:1–18, 1 Kor. 13:1–8, Jer. 17:9, Kol. 1:1–12, 1 Pet. 1:4, Saam. 119:105, Isa. 30:21.

> <p>Bible cangken</p>
> “Nanmah ih hnatuan tha a tuan thoktu cun Jesuh Khrih ih ni tiang a tuan peh vivo ding a si timi rori hi ka zum hnget a si” (Filipi 1:6, NKJV).

Pual cun a cakuat ngannak ahhin cibai buknak – le lungawi thusimnak a telh ringring theu. “Pathian kan Pa le Bawipa Jesuh Khrih hnen ihsin zangfahnak le daihnak cu nan hnenah um hram seh. Pathian le kan Bawipa Jesuh Khrih ih Pa hnenah lungawi thu kan sim” (Kol. 1:2, 3, NKJV).

Kannih khal Paul vek ciah in lungawinak ding tamzet kan nei. Pathian zangfahnak tonnun tamzet kan nei, van mi pawl ih an theihthiam thei lo ding khawp tonnun kan nei. Cuvek ciah in Pathian duhdawtnak ihsin a luangsuak mi ruahsannak le remdaihnak pawl cu Pathian ih daihnak laksawng a si.

Milai dinhmun khal in vun zoh hnik sehla, midang parah lungawinak kan langter theu, mi thil kan tuahsak asile, kan parah an lungawinak langter ve dingin ruahsannak kan nei cio. Nu le pa pawl cun an fale pawl in Pathian an duhdawt ih, a tu rori a si lo hmanah nikhat nini ah an fale hrangih an pumpek awknak pawl hmuahhmuah an run theihthiam theinak dingah thlacamnak an neih sak theu. Asinan minung kan si vekin, thil diklo tamzet kan tuah ih, cumi ihsin zirlai kan zir theu (a silobik in kan zir tengteng a tul).

Tuizarh ahcun Filipi le Kolose kawhhran hnen ah Paul ih lungawi thu simnak le thlacamnak tongkam a hman hmaisabik mi pawl kan thlir ding. Cumi cun kan thlacam nun ahhin thazang cahnak le phurnak in pe ding a si.
