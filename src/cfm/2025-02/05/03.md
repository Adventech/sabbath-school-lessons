Babilon mi pawl hnen ih sal ih hruai an si lai fang a si ih, Profet Daniel hmang in Pathian cun A minung le leitlun ramuknak pawl karlak ih laicin awknak a relfiang mi pawl kan nei. A minung pawl cu mahte ukaw mi an si nawn lo: An duh hrilnak ih rahsuak mi a khawm rero a si (An hnen ihsin ziangsaw kan zir thei ding?)

`Daniel 2:31-35 siar awla, cumi cun caan netnak tiang a um dingmi leitlun san thuanthu a lo hmuh ding. Himi mangbangza simcianak ihsin a thupi zet mi thudik ziangsaw kan zir thei ding?`

Kum zabi 19 cem hrawng ahkhan mi tampi cun minung thansohnak lam ah rinsan awknak thar an nei. Thimnak ah Paris Exposition (1900) cu hmailam thuhla ih ruahsannak tarlang mi ropi zet a si. Kan fimzir thiamnak le theihnak lam ih kan thansoh pi mi hmuahhmuah thawn minung ih buainak rapthlak zet pawl cu cemter an si thluh leh ding. Kum zabi 20 ih mipi an luh tikah thuruahnak tampi lakah himi duhthusam hmuhfiangnak– Minung ih famkimnak le thinlung thiltitheinak huham cun – minung hrangah caan (santhar) thar mangbangza a run thlenpi thei a si timi ruahsannak ropi zet an nei.

Ziangkhal sehla ralpi veikhatnak cun cui duhthusam sunmang pawl cu a run vaw siat thluh ih, kum zabi 20 a cem zawng ah raldoawknak ah mipum 200 million lai an nunnak an hloh. Thiamnak lamah kan rak thangso zet tla si men thei, asinan Pathian lam ahcun kan thangso lo. Dr. Martin Luther King Jr. ih a felfiang dan ahcun lam hmuhtu kan nei, asinan minung hruaisualtu a si. Cumi cu thil tihnung um zet a si.

Simcianak zirtu tlawngta tampi cun Daniel 2 ih a ummi ‘thir’ cu a man sangbik ihsin a tlaniam a si timi an theithiam: Sui man cu ngun ah, ngun man cu daar ah, cule daar man cu thir ah, thir le tlak ih a thlen tiangin man a nei nawn lo deuhdeuh.

Charles Darwin le Karl Marx le kum zabi 19 ih thuruah thiam zet pawl cun minung cu ziangsimaw ihsin hung thangso vivo mi an si – suahkehnak thleng aw so vivo mi an si tivek thawn kan thinlung neh an tum. Khalam zawng ahcun minung hmuhtonnak hi thangso zet khal sehla (tisa thil lam tal ah), a tu ih ram ukawknak, remdaihnak, kilhimnak le neihnunnak thuhla ah hi leitlun hmailam caan a thlir thei ngaingai tu zosaw um?

`Jesuh cun ralrinnak a pe ih, “ ‘Cun doawknak pawl le doawknak thu thang tla nan thei ding… Ziangahtile hnam khat le hnam khat an doaw ding ih, ram khat le ram khat an doaw ding: cun hmun dangdang ah paam a tlung ding, natsia a thleng ding ih, ling tla a hnin ding’ “ (Matt. 24:6, 7). Himi ralrin peknak um cing in ziangtin saw hivek thuhla ah ralrinpek kan rak si zo timi va theihnak ihsin hnangamnak kan neih thei ding?`
