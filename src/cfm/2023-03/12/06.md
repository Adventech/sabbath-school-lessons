---
title:  Sual thiltitheinak dodalnak
date:   14/09/2023
---

`Efesa 1:21, Efesa 3:10, le Efesa  6:10–20 ih a tarlang mi thlarau lam thiltitheinak sual hmin ci dangdang a tarlang mi ah Paul ih tumtah mi cu ziang a si ding tin na ruat?`

Paul in “Kan zuam tentonak” (Efesa. 6:12, NRSV) tin a run rel mi hi Grik tongfang ahcun pahnih karlak ih buan awknak(pale) ih zuamawknak a si. Buanawknak cu doawknak hrangih timlamnak tha bik a si ruangah himi cu ralkap pawl an doawk tikah ralthuam le ralthuam, kut le kut doawknak timi a mawi zawngih simfiangnak a si. Paul cun zumtu pawl cu a takram in sual thiltitheinak pawl cu do dingin a duh zet.

Hitawk ah thupi tete in pek suak a si:

| Efasa 1:21 | Efasa 3:10 | Efasa 6:12 |
| --- | --- | --- |
| Thu neitu zaten | Thuneitu | Thuneitu |
| (zaten)Thuneihnak | Thuneitu | Thuneihnak |
| Thiltitheinak (zaten) | | Tui umnak thim tlun ih vanlam thiltitheinak |
| (Zaten) Ukawknak | | Van lam hmun ih um thlarau sual pawl |
| Hminsakmi hmin tinkin | | |

A tlun ih simfiang mi(“hminsakmi hmin tinkim” Efesa 1:21,LEB; “Van lam hmunsang ih um thlarau sual pawl”Efesa. 6:12, ESV)  ah Paul cun khavek sualnak hmuahhmuah le sual thiltheitheinak nasa zet pawl cu Khrih in a neh thluh thu a sim (Efesa. 1:21). Ziangkhal sehla ziangvek raldoawknak khalah khatlam sawn tu kha nautat ciamco dingin kaihhruai awknak cu a tha dah lo. Paul cun kan raldo mi pawl cu minung ral an si lo, asinan “van hmun pawl ah sualnak le thlarau sual pawl an si” tiah a ti (Efesa. 6:12, NKJV), annih pawl cu ralkap hotu, khawse lal ih hruaimi pawl an si (Efesa. 6:11). Ziangkhal sehla, doral thiltithei zet a um thuhla ah ralringnak pek kan si lai ah, anmai ruangah thinphang ih um ding kan si lo. Raldo awknak ah Pathian cu kan hnen ah a um a si (Efesa. 6:10)cule Anih cun kan tulmi ralthuam pawl in pe. Amai ralthuam, “Pathian ralthuam” (Efesa. 6:11; zohtlang aw, Isa. 59:15–17) rori in pe a si. A diknak, dingfelnak, daihnak, zumnak le rundamnak, le thlarau thianghlim cu kan hnen ah in pe a si(Efesa. 6:13–17). Kan hmai ih Pathian a feh ruangah amah in in pekmi kan ralpawl kan nehnak ding ralthuam thawn kan sung thei hrimhrim lo ding. (Rom. 16:20; 1 Kor. 15:23, 24; 2 Thess. 2:8).

`Himi sual thiltitheinak pawl, kanmah, kanmah rori ih kan do neh thei lo mi pawl cun in zirhmi cu Jesuh, hivek thiltitheinak pawl hmuahhmuah ai ih cungnung sawn lawng hman siloin, anmah pawl a neh zotu cu ziangtluk in kan nun ah kan tul timi ziangvek in saw in zirh?`
