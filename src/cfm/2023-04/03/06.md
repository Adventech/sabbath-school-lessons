---
title:  Na umnak ihsin thok aw
date:   19/10/2023
---

`Siar ding: Tirh 1:8. Leitlun ah a thu theihpi tu le Pathian hnatuannak neih a si lai ih Jesuh ih kaihhruainak hi ziangsaw a si?`

Himi kaihruaiawkdan cu Jesuh ih tuahsuak mi rori a si, cumi cun a dungthluntu midang hnenih thuthangtha va phuangtu ding kan si vekin ziangtin kan um ding timi in hmuh a si. Thutak sim hi midang hnenih an diklonak va pom ter ciamco lam a si lo ih,  vancungmi pathum thuken, Thup 14:6-12 sungih ummi vekin Jesuh va theihter sawn hi a si.

Ziangkhal sehlah Tirth 1:8 ah Jesuh ih kaihruaiawknak tongkam a  um.

Pakhatnak ah, “Jerusalem ah ka thu theihpitu nan si ding” (NIV). Kan hmuh zo vek in(Asinan relnawn sal rero tlakza thil a si): Taksa rori ih kan umnak hmun kip ah a thutheihpi tu si ding kan si. Himi ih tel thei mi cu kan inn, kan kawhhran, kan inn hnen le kan pawlkom pawl an si. A hmaisabik ah kan umnak hmun kip ah a thu theipitu kan si a tul, a hramthok ih in retnak hmun – inn lole hnatuannak tivek khalah, kan kiangnai ih um pawl hnenah a thutheihpi tu kan si a tul rori a si. Cumi ahcun kan laicin lole kan sungkhat, kawhhran mipi pawl, kan tlawngkai pi pawl, kan venghnen pawl le kan pawlkomnak hmun pawl khalah an tel thluh.

Khui maw caan ahcun mipi pawl cu hmun hlazet, ram hla zet ah feh in a thutheihpi tu si an duh ciamco. Asinan an kiangkap ih um mipi pawl hnenah a thutheihpi tu an si fawn si lo. Kan umnak hmun ihsin kan thok a tul, cumi ihsin Pathian ih in kaih hruainak ah feh ding fang kan si.

Neta sawn ahcun, “Judah ram hmuahhmuah le Samaria ram, le leitlun dengkhawr tiang” (Tirh; 1:8, NIV)ti a si. Himi ah Jesuh cun thutheihpi tu sinak cu phunhnam leng ih um khal a tul timi a run rel fiang. Kan umnak hmun ihsin hmuhdang deuh, pawlkom dang deuh, miphun dang deuh le biaknak sakhua dang deuh ih um dingin kawh kan si men thei. Hnam cikhat lole tong cikhat umnak mipi lakih kan um ahcun hnam nunphung lamah harsatnak malte lawng kan neih ding ruangah kan hrangah a thutheihpi tu si cu a olsam men thei. Leitlun hmun hrekkhat ahcun kawhhran dinthoknak ahhin hnam pakhat lawng tivek an um theu.  Ziangkhal sehla, Jesuh ih thupek ropi zet cun A thutheihpitu kan si vekin kan umnak hmun ihsin suak in, mipi hrangih kan theihnak ziangkim suah hi a thupi zet tin in sim. Mipi pawl ih tul mamawh mi bik thucah cu Jesuh hi a si.

`Sawmngiarnak: Na umnak khawtlang ih tulsamzet pawl, kawhhran in thuthangtha phuan dingih an hnen ih a thleng hrih lo mipawl kha theih tum awla, hminsin hai aw.`

`Sawmnak: Hmailam tikcu caan ah a tul mamawh zetmi pawl hnenih Pathian hnatuannak tikcu caan tha a thlen theinak dingah thlacamnak nei thok aw.`
