---
title:  Ram cu laksawng a si
date:   24/11/2025
---

`Siar ding: Suahlannak 3:8; Puithiam Hnatuan 20:22; Puithiam Hnatuan 25:23; Mipum Siarnak 13:27; Daan Peksalnak 4:1, 25, 26; Daan Peksalnak 6:3; le Saam 24:1. Pathian le Israel mipi pawl le Tiamcia Ram karlak ah a hleice ih laicinawknak ummi kha ziangsaw a si?`

A hramthok lam ahcun ram cu tisa takin miphun pakhat hrang a si. Miphun an umnak thawngin cui hmun cu phunhnam nungphung le hnatuannak tivek khal a um thluh. Sal pawl cu hrihram nei lo an si ih, umnak ding hmun khal an nei lo. An tuan rah cu midang in nuam zetin an rak cen. Ram nei timi sullam cu zalennak tinak a si. Hrilcia miphun pawl theihfiangnak cu an umnak ram thawn pehtlaih awknak hnget an neihnak ahhin a si.

Pathian, Israel le tiamcia ram karlak ahhin laicin awknak hleice a um. Israel pawl cun ram cu Pathian laksawng vekin an dong, mai diknak covo lak sak theih lo mi vek in an dong ve lo. Hrilcia miphun pawl cun Yahweh thawn laicin awknak thukam ih an um kulhsung le tihzah zetin thukam an thlun kulh sung cu ram hi anmai ta a si ding. Phundang in kan sim asile Pathian thlawsuahnak tel riai loin ram le ram ih thlawsuahnak an dong thei lo ding.

Cuti rual in, Israel pawl in Pathian an theihfiang thei deuhdeuhnak ding ah ram pek in an um timi hi thudik a si. Ram sung ih an umnak cun Pathian rinsan dingin, thutiam thlun dingin le rinum zetih um dingin a theihtersal ringring a si. Pathian ih hmainornak tel lo sehla, Israel miphun simaw, ram khal hi a um zik lo. Pathian cu anmah pawl hung umnak bulhram bik a si. Israel pawl Izipt ram ih an um lai ahkhan Nile tiva le tidai ngahnak siseh, hna harsa zet an tuannak ihsin an nun theinak dingih an tul zet mi thlaihnah thlairah dangdang a rak pe. Kanaan ahcun a danglam thluh. Lo rawl tamzet an neih theinak dingah van ruah ah an thum aw, van boruak kilkhawi theitu cu Pathian lawng a si. Curuangah rampi cun Israel pawl khan Pathian ih an thumawk a tulzia thu a theihter ringring a si.

Israel pawl in Yahweh hnen ihsin rampi cu laksawng vekin rak dong khal sehla, a netnak bik ih ram neitu ngaingai cu Pathian a si thotho. Leilung pumpi neitu a si vekin (Saam 24:1), Yahweh cun Israel pawl hnen ah rampi pek ding, asilole laksak theinak a nei a si. Pathian cu ram neitu a si ahcun Israel fale pawl le, a huapkau zawngih kan rel cun minung pawl hmuahhmuah hi mikhual le khualtlawng mi men kan si, asilole santhar nun fehdan ih kan thlir ding cun, kan zaten Pathian ih ram/ leitlun ahhin reipi rak um zo mikhual men fang kan si.
