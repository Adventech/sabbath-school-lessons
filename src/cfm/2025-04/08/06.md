---
title:  Khawruah thlengnak
date:   20/11/2025
---

`Zumnak lam ih paraltha pawl ih nun zohthim ding vun ruat hi thlarau nun thansohnak ding ih thil thupi zet a si. Himi tikcu caan ahcun a netabik ih kan zohthim dingmi cu Jesuh Khrih – A nun le A thuzirhnak hi a si. Ziangtin saw Jesuh nun kan ruahnak cun in thleng danglam thei? Zoh aw, Heb. 12:1, 2; 2 Kor. 3:18.`

Los Angeles, California Phunsang tlawng ih neuroscientist (thinlung{thluak} lam zirnak) zirhtu Marco Iacoboni cun thinlung lam hnatuandan a zirfiang. Himi cell fate zet vak kual rero pawl cun thil danglam deuh kan tuah caan - midang hnihter, lole kan kuah caan le midang pawl in cuvek ih thil an tuah lai kan hmuh tivek ahhin nasa zetin hna an tuan cuang a si. Himi thinlung lam hnatuandan cun hmuh mi le tuahmi karlak ah ah danglamnak mal deuh in tuah a si.

Ellen G. White cun Jesuh ziaza zohthim ding a thupit zia a rel: “Jesuh lam kan hoi tikah Pathian kan thlirdan a danglam deuhdeuh ih, a tleu mawi deuhdeuh, Jesuh kan zohthimnak ihsin kan danglam a si. Kan minung pi kan duhdawt ih thiltha kan tuahnak pawl cu kan suahpi nunphung ah an hung cang ding. Pathian ziaza thawn milaw zet ziaza neihnak lamah kan thangso ding. Amah bang sinsinnak lam ah kan thang so ding ih, Pathian kan theihnak khal a kau sinsin ding. Van lam mi thawn pawlkom awknak ah kan lut sinsin ih, kannih cu kumkhaw fimnak le theihnak lamah zetih dong dingin huham cakter mi kan si.”—Ellen G. White, Christ’s Object Lessons, p. 355.

`Siar ding: Rom 12:1,2. Kan nun ih dokalh aw rero mi pahnih cu ziangsaw a si? A dik sawntu lamah kan um a si timi hi ziangtin kan fiang thei ding?`

Rom khawpi hrang ih cakuat mi bungcang kaihtawinak ah Paul cun kan nun in buaiter tu thil pahnih thuhla a rel. Khat lam ah kiangkap leitlun ih thilri tamzet pawl cun nitin kan nun in ciahneh ih, cumi cun kan sungah hna tuan in, kan nun thleng in tum.

Cui thil pawl rak dolet dingin pangang cu phengpelep mawi zetih a can vek ciah in Thlarau Thianghlim cun kan sunglam nun khal in thleng danglam thei a si. Asinan cumi cang suak thei dingin Pathian hnenah kan pumpek ih, kanmah ih a rak thok mi cu theh suak dingin kan ngen a tul a si (Filipi 1:6). A netnak ahcun thlarau ih feh thei dingin tikcu caan ziangmaw zat cu theihfimnak nei in duhhrilnak kan tuah a tul.
