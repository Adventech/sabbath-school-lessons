### Tuizarh zirlai hrang siar ding

Josh. 5:1–7; Suah. 12:6; 1 Kor. 5:7; Josh. 8:30–35; Daan. 8:11, 14; Heb. 9:11, 12.

> <p>Bible cangken</p>
> “Pathian uknak le a dingfelnak cu hawl hmaisa sawn uhla, hi thil hmuahhmuah hi pek beet nan si ding a si” (Matthai 6:33).

Tuizarh ahcun Ram Tiam sungih Israel pawl an umnak ih thil thupi zetzet pawl kan zoh ding. Israel in Bawipa hnenih an hlanawknak, khuimaw caan ih tihnung um zet dinhmun ih an um caan tivek pawl khal kan zir ding. Doral umnak ram kiangkap ih Israel pawl an um laiah Joshua cun serhzim tan um dingin (Joshua 5:-19); tihphannak hmai ih an um lai ih lantakpuai hmang dingin (Joshua 5:10-12); ralnehnak an neih rero lai ah biaktheng tuah ih Pathian bia dingin (Joshua 8:30-35); le Israel hnam 7 pawl in an roco dingmi an ngah hlan ih (Josh. 18:1, 2) Bawipa ih biakbuk tuah dingin thutluknak ngam huai zet a tuah.

Kan nun buai zet ahcun kan nun ih a ra thleng mi manhlap thlak zet thil kan ruat theu. Pathian hnenih kan pumpeknak kan tharthawhsal theinak dingah le kan hrangih a tuahmi le nitin in tuahsakmi ruangih kan lungawinak langternak ding caan tha hi kan hngilh leuhleuh theu. Zing le tlai ih Pathian biaknak neih le sungkua ih thlacamnak neih tivek pawl cu kan nun buaithlak zet le thil olsam zetih tuah duhnak thawn cun an milaw lo zet vekin a lang. Asinan thinlung ngaingai te ih vun ruat ahcun Pathian le kan duhdawtmi thawn caan hman tlang hi kan tikcu caan hmanmi ih a thabik mi lamzin a si.
