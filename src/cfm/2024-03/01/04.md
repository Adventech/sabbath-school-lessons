---
title:  Thucah kengtu
date:   02/07/2024
---

`Siar ding: Maraka 1:1-8. Himi bungcang sungih um mi hi zo pawl saw an si, ziang thu an rel ih an tuah?`

Himi bungcang pawl ahhin mi pathum thupi zet an lang : Jesuh Khrih, Marka 1:1 in a khih hmuh ih, Pa Pathian a si ih (Marka 1:2 tongkam zoh awla), cule Baptist tu Johan, thucah kengtu le himi cahmai netabik lam ih hmunram thupi zet runluahtu a si. Marka 1:2,3 ahcun Thuanthu in ziang a run feh pi ding timi Thukam hlun sungta thuhla la tahrat in Marka cun a run sim. Marka ih a run laksuah mi cu Bungcang pathum simfiangnak ihsin ra suah mi an si: Suahlannak 23:20, Isaiah 40:3, le Malakhi 3:1.

`Siar ding: Suahlannak 23:20, Isaiah 40:3, le Malakhi 3:1. Himi bungcang pathum in ziangmi bangawknak saw an neih?`

Suahlannak 23:20 cun Israel pawl Kanaan ram ih hruai dingah an hmai ah vancung mi Pathian in a thlah ding thu a run khihhmuh. Isaiah 40:3 cun Pathian in amai lamzin sialnak ding hrangih nelrawn lamzin ih a um thu a run tarlang ih, Malakhi 3:1 cun Pathian hrangih lamzin rak sialtu dingah a hmai ih thucah kengtu a um thuhla a rel lang. Himi bungcang pathum pawl cun khualtlawnnak lampang an sim thluh.

Isaiah cabu sungih um mi cun tipil petu Johan ih hnatuan ding tamzet an run langter, cule Bawipa ih lamzin khal an run hmuitin thotho. Thuthangtha Marka ahhin Bawi Jesuh cu khualtlawnnak ah a um. Himi khualtlawnnak ahhin lungawinak a um ter ih, himi khualtlawnnak cun thinglamtah lam ah le kan hrang a raithawinak lam ah a hruai thleng.

Asinan thinglamtah a thlen hlanah thil a hung cang dingmi tamzet an um. Khualtlawnnak cu a hramthoknak fang a si. Marka cun cumi pawl cu in run sim thluh lai ding. Marka 1:2,3 ih um thukam hlun ihsin thuhla laksuah mi ah thum aw in tipil petu Johan cun sualsirnak nei in, sual ihsin kirsal in, Pathian hnen ih kir lehsal dingin kawhnak a nei (Marka 1:4). Hlansan profet Elijah vekin a thuamaw ih, (zoh kop aw, 2 Siang 1:8) Marka 1:7,8 ah amah thluh hnu ih a ra ding pa cu amah hnakih a nasa sawn ding thuhla a sim. A thusimmi ahcun anih cu a ra dingmi hmaiah kun tahrat in a kedam hri phawrh saktu ding hman ih a tlak lo thu a simmi cun Jesuh a cawisan zia a langter.
