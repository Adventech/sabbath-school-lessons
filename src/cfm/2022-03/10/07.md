“Nunnem le zawitheinak neitu Khrih sungkhum nun neih ahcun a harsatuk mi ton a tam lo. Kan Bawipai tangdornak kan nei ngaingai ahcun nitin hnaihnoknak kan ngaihsak lo dingih thin nuam lonak a reh ding. Cule tlunlam ih lam hmuhsak mi kan pan thei ding. Mah le mah uk aw theitu Khrihfa in tihzah upattlak nunze mawi a langter. Eiruknak le nunrawngnak ngamzet le hnangam tei’ tuahtu a thinlung duhnak thunun lotu cu amah ah Pathian in nunzemawi le tha a pek mi hnongtu a si. Jesu Khrih dungthluntu hrangah tangdornak thinlungput cu nehnak petu cahnak thazang a si. Anih cu van rorelnak hmunpi thawn an pehzom aw.” Ellen G. White, The Desire of Ages, p. 301.

**Reltlangding**

`Hnaihnoknak le harsat tuarnak ah tongdornak thinlungput in ziangvekin so in khaisang? Ziangvekih umdan tangdornak in hivek tuah theinak a suahpi tiah na ruah?`

`Bulpak na nundan ah, tangdornak le nunnemnak umdan ziangvekin so a lang? Cuih pawl cu ngaihsak a hlawh in upat a tong maw or ziangvek so a si? Na nundan ih lo hnaihnoknak ziang so na tong? Tangdornak le nunnemnak nunze nei thei lo dingah ziangtin so a lo buaiter?`

`Tuisan minung pawl nunziaza ihsin zohthimtlak nunnemnak le tangdornak a um maw? A um ahcun zote pawl so an si? Himi nundan ziangvekin so an langter? An hnen ihsin ziangso na zir thei?`

`Tangdornak le nunnemnak hi ziangruangah tontaihnak vekah kan hmu theu?`

`David in Bawipa cu ka beunak a si tiah a ti mi kan hmu. Asile khami a beu dan ziangvek so a si ding? Khami beunak cu ziangvekin so a langtheu? Phundang ih sim ahcun, Kawhhran kan si vekin anhai pawl hrangah beunak ziangvekin so kan tuah ding? Tualsung kawhhran ah anhai pawl beunak ziangvek so nan tuah? Anhai pawl beunak hmun ah ziangso nan bawm?`
