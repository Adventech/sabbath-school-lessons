---
title:  Zawitheitu Pathian
date:  04/09/2022
---

`Rom 15:4, 5, siar aw. Kan hmuh thei mi ziangso a um?`

Kan thinlung ih kan duh zet mi or kan hnen ih thukam mi a thleng hrihlo mi thuhla ah a tlangpi  zawi theinak kan nei lo. Kan duh ih kan hngakhlap mi pawl kan ngah tik lawngah lungawi mi kan si. Kan duh canah kan duh mi kan ngah thei lo ruangah kan thin a tawi ih kan zawi thei lo. Hivek dinhmun ah Pathian rinsan in hnangamten kan um thei lo.

Hngak can hi rei kan ti theu. Hebru pawlin tong pakhat an neih mi cu “thinsau zetin hngak aw” (Saam 37:7, NKJV). Hebru tong an lehlin sin mi pawl cu “hngak theilo ih thintawi theinak” “thin thir thehnak” “taksa tiang ih thir le thia theinak” “tuar thei nawn lonak”  “riahsiat theinak.” Zawitheinak zir hi a ol lo, himi hi hrasatnak tuar can ih thil um dan an si.

`Saam 27:14; Saam 37:7; le Rom 5:3-5, siar aw. Ziangpawl so kan hnenah an sim? Zawitheinak in khui lamah in hruai?`

Thil pahnih kan hngak lai ah thinlung ih vei sawn mi kan nei theu. Beiseinak thawn kan hngah mi cu kan thinlung ih kan vei mi or Pathian kut sung ih a um mi kha kan beisei thei. Thil pakhat kan hngah tikah a danglamnak cu ziangtluk ih rei kan hngak ding timi or kan hngak lai ah thinlungput ziangvek kan nei timi a si. Bawipa kan rinsan ih amah thuthu siseh kan ti tikah kan thinlung duhnak cu a hnenah kan ap ahcun, kan hrang a thabik a tuah ding tiah kan rinsan ding. Kan hrangah a thabik a si ahcun harsatnak a um khal le a poi lo tiah kan ruat theu.

`Beidong theh ih na hngah mi ziangpawl so an um? Pathian ih can rikhiah mi sungah le Pathian kut sungah pumpekaw thluhnak ziangvek in so na tuah thei ding? Bawipa hnenah pumpek aw pumhlumnak ih a thupek vekih um dingah na thinlungput le na nun ziaza ih thlun dingah thlacam aw.`
