---
title:  Ruahbetding
date:  12/08/2022
---

“Riahsiat ninghannak kan tuar can ah beidong in kan um theu. Hivek harsatnak can ah Pathian in leitlun ih a hrin mi a fa le pawl a zangfah ringring timi zum a har zet. Harsatnak le buainak kan ton ni le can ahcun thih kan hril sawn theu. Cutikah Pathian rinsannak an thlah ih rinhmainak le zum thei lonak salah an tang. Kha lai canah Pathian zangfahnak zarah zumnak thar le nunnak thar kan nun ihsin a hungsuah dingah van mi pawl in kan nun ah tlang hnakih hngetkhoh sawn ih dingsuak thei dingah in helhkam rero ti mi ruahnak nei in kan thei fiang thei maw.” Ellen G. White, Prophets and Kings, p. 162.

**Reltlangding**

`Riahsiatnak kan tong ih beidong in kan um cio tiah Ellen G. White, in a sim. Khami can kan paltlang lai kha kan thei cio maw? Beidong le vansang ih kan um can ah ruahsannak palai tha si thei dingah ziangvekin so kan zir thei ding?`

`Tlawngkai nihnih thusuhnak netabik kha zirlai zirnak ah reltlang uh. A hlan hnakin Pathian in a lo naihsawn timi theihfiangnak in na nunah danglamnak ziangvekin so a tuah?`

`Zirlai zir canah Job 38-41, sung kha ringpi in siar uh. Pathian pianhmang ziangvekin so a lang? Ruahsannak le thathonak ngah dingah ziangso na zirsuak? Himi hmuihmel langter dingah Sabat nizarh hi a remcang thei maw? Pathian ziaza le sinak a hlan hnak ih cingkeng thei dingah ziangvekin so a lo bawm?`

`Nun thlengdanglamnak cu van ihsin a ra ding tiah an ruahsan. Himi ih asullam cu bulpak nunah ruahsannak ih thlacmnak a thlenpi tinak a si. Cuih pawl cu ruahsannak tharsuah theinak dingah can tampi nasa zetin thla an cam ih pai cunco in an feh. Midang ih buainak hlo theinak dingah le ruahsannak an neih theinak dingah khami hnakih thasawn na tuah thei mi ziangso a um?`

`Mi hrekkhat pawl cun hlanlai an nun hlun ih harsatnak le beidong vansang ih an umnak pawl thei dingah suh an duh theu. Khai minung kha khuilam so a herter? Beidong le vansang ih kan um laiah ih tonnun cu midang bawmnak ah ziangtin so kan sim thei ding?`
