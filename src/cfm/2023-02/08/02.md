---
title:  Thuthennak, sersiamnak le tuanvo
date:   14/05/2023
---

Kannih hi cithlah zomkhawm aw ve maimai kan si ahcun ziangmaw thil ruangih um le Africa ram ih mei nei lo zawngrual hnakin hman in zianghman kan si cuang ziklo, cule hrinnun hin malte lawng sullam a nei ding. Leitlun tiih kawhmi minung tingthawngtel ummi lakih mi pakhat ve maimai kan si ahcun, hringnun hin a tumtah mi a hloh ding ih, nung fangfang men kan si ding. Cuvek men ih um loin Bible ih sersiamnak cun nunnak ding san le vang le hringnun hman dan ding in sim. Pathian ih sersiammi kan siih, curuangah kan thiltuahmi ah a mi siar tlak kan si a tul. In sersiamtu Pathian cun tuan ding tuanvo in pe a si. “Laicin awknak ngai thupi zetnak” leitlun khalah Pathian cun thildik cu a dinpi thotho a si.

`Siar ding: Thuphuan 14:7, Rom 14:10, le Jame 2:8-13. Thuthennak ah tuanvo nei le tuanvo la tivek khin ziangsi a kawh hmuh? Ziangtin saw thuthennak, Pathian thupek pawl le cibai buknak hi an zom awk thei thluh?`

Thuphuan 14 ih um van lai ih a zuangmi vancungmi pathum ih thucah cun, “A thuthennak caan a thleng zo”(Thup 14:7) ti thu an than. Duhhril theinak nei dingih Pathian sersiammi kan si ruangah thutluknak kan tuah dinghi kan hna a si. Cithlah sawng awknak satliah menmen, kan kimvel hrang lawng ih hmantlak menmen mi kan si cun kan thil tuahmi a tawm sawn cu kan khap thei lo mi, hranhram ih tuahmi vek ah ruat an si ding.

Asinan thuthennak cun nuntu ziaza ih tuanvo lam a kawh hmuh sawn. Hi leitlun santhuanthu ih buainak, thuthennak tikcu caan ahhin, Pathian cun kumkhaw Eng ih um dingah thutluknak tuah dingin in sawm a si. Vancungmi pakhatnak cun nasa zet in, “Van le lei, le tifinriat le cerhti pawl a tuahtu cu bia uh”(Thup 14:7) tiih ngennak cun Cibai buknak ih bulhram bik cu Pathian sersiammi kan sinak hi a si timi a theihter a si.

Cuti lai ah, Nisarihnak Sabbath kan ngai thupit hin Jesuh cu kanmah in tuahtu biaktlak a si timi kan tarlang a si. Cumi cun hringnun famkimnak ding hrangih pekmi thukham 10 Daan, Pathian thawkkhummi kaihhruaiawknak kan rak cohlang timi a tarlang a si. Ziangahtile Daan cu Pathian uknak ih thum awknak le a mizia phuangsuahnak a si. Cumi cu thuthennak ih tahfung khal a si. Sabbath thupek ih kan rinumnak cun thuawih zetih hringnun hmang dingin pumpe aw in kan um timi a langter.

`Sersiamnak ih in ciahnehnak hin kan mizia ah ziangvek theihthiamnak saw a neih? Nitin ten duhhrilnak kan tuah dingmi ah kan kimvel le laicin pawl hin ziangvek pehzom awknak saw a neih? Pathian zangfahnak thawngin a thokih kan duhlomi mizia pawl kha ziangtin saw kan neh thei ding?`
