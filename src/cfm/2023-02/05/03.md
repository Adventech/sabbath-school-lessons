---
title:  Pathian zangfahnak le thuthennak
date:   24/04/2023
---

Thinglamtah le thuthennak cun Pathian ih diknak le zangfahnak an langter veve. Daan buarnak cun misual thihnak a ngen a si. Diknak cun, “sualman thihnak” timi a tar lang ih, zangfahnak cun, “Pathian laksawng cu kan Bawipa Jesuh Khirh ah kumkhaw nunnak” timi a tarlang thung(Rom 6:23 NKJV). Pathian daan hi thleng theih, lole bal theih a si cun Jesuh a thih hi ziangtilam khalin a tul zik lo. Khrih thihnak cun daan hi kumkhua in hnget ter ih, cule thuthennak ih thumawknak cu daan hi a si.

`Siar ding: Thuphuan 20:12. Ziangtin thuthen kan si ding? Kan hnatuan that mi pawl hi kan rundamnak thawn ziangsi pehtlaih awknak an neih?`

Kan hnatuannak hin kan duhhrilnak le Pathian ih kan rinumnak a pholang a si. Efisa 2:8,9 ih ummi vek a si le, “Rinnak ruangah ngilneihnak zarah rundam nan si… Tuannak in asilo, cutilocun mi an porh aw pang ding” tiah a ti. Asinan Khrih ih rundam kan si tikah, anih cun in thleng danglam. “Kannih cu amai’ kut suak, hnatuan tha tuan dingin Khrih Jesuh ah in sersiam mi kan si”(Efisa 2:10)

Thlarau thianghlim ih cahnak inpek ruangih kan thiltuah that mi hin in rundam lo ih, kan rinnak cu a taktak a si timi a theihter sawn hi a si.  Pathian thuthennak netabik cun lemcangnak hmuahhmuah, ngaithupit awknak hmuahhmuah, thudik lo hmuahhmuah a hlip thluh ih, kan sinak thuk zetah a luhpi a si. Thuthennak ah Pathian hmaiih kan dinhmun langter a sinak ahhin Ellen G White cun Thuthangtha le thuthennak a fehtlangnak thuhla hi theithiam dingin nasa zetin in bawm a si.

“A thianghlim lopi hnipuan thawn Pathian hmai ah dingvek ih a langmi Pathian ih minung pawl ih a hmuhsak mi cu hnuaidornak le thinlung takten hawlsuak ih Amai hmin a phuangsuak tu pawl an si. Thutak thlunnak ihsin thlarau ti thianghlim aw tu pawl cu mah le mah hnuaidor aw zet nun an nei ding. Jesuh ih mizia an hmuh fiang deuhdeuh tikah a hmulhmel bangnak neih an duh deuhdeuh ding ih, anmai thianghlimnak pawl cu an hmu mal deuhdeuh ding.  Asinan kan sualnak dinhmun kan vun theih tikah, Kan dingfelnak, kan thianghlimnak, le tlanmi kan sinak pawl cu Khrih ah a um timi kan theih a thupi zet. Jesuh amah lawnglawng hi kan ai awh theitu umsun a si. Kan thiamnak ah siloin, amai’ par lawngih thumaw mi thubuai hmangin phiartu cu daiziar in a ret thei a si.”—Testimonies for the Church, vol. 5, pp. 471, 472.

`A tongkam ah thuthennak le thuthangtha ih thenawk a theihlo thu ziangmi saw na hmuh? Himi thuthangtha le thuthennak ih pehzom awknak ihsin na hrangah ziangmi beiseinak saw na lak thei ding?`
