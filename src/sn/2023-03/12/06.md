---
title:  Kurwa naMasimba Akaipa
date:   14/09/2023
---

`Unofunga kuti chinangwa chaPauro ndechipi pakushandisa mazita emasimba emweya akasiyana siyana sezviri muna VaEfeso 1:21, VaEfeso 3:10 naVaEfeso 6:10-20?`

Pauro anotsanangura “kurwa kwedu” (VaEfeso 6:12), achishandisa shoko rechiGiriki remakundano pakati pevatambi vomutsimba (pale). Sezvo mutsimba waionekwa segadziriro yakanaka yehondo, ndiyo tsananguro yakafanira ye kurwisana chombo-ne-chombo uye maoko-ne-maoko kunoitika kana varwi vasangana. Pauro anosimbisa huchokwadi hwekurwa kwavatendi nemasimba akaipa.

Hawai mazita anopiwa kumasimba aya:

| VaEfeso 1:21|VaEfeso 3:10 | VaEfeso 6:12 |
|---|---|---|
| vabati vose (hutongi hwose) | vabati | vabati |
| vukuru (hwose)| vanesimba | vanesimba |
| masimba (ose) | | madzishe enyika erima rino |
| vushe (hwose) | | mweya yakaipa yedenga |
| mazita ose anorehwa | | |

Mutsananguro yake yakatambanudzwa (“mazita ose anorehwa,” VaEfeso 1:21, “mweya yakaipa yedenga,” (VaEfeso 6:12), Pauro anopupura kuti masimba ose akaipa ari pasi paKristu (VaEfeso 1:21). Zvisinei, muhondo ipi zvayo, harisi zano rakanaka rokutarisira pasi simba revamunorwa navo. Pauro anoyambira kuti hatirwi nevanhu, asi “nemweya yakaipa yedenga” (VaEfeso 6:12), inotungamirirwa nomukuru wehondo wakachenjerera pakunyengera, diaborosi (VaEFeso 6:11). Zvisinei, kanapo tichifanira kugara takangwarira masimba aya, hatifaniri hedu kuvhundutswa nawo. Mwari anesu muhondo iyi (VaEfeso 6:10) uye akatipa zvombo zvakanakisa, nhumbi dzose dzokurwa dzake, “dzaMwari” (VaEfeso 6:11; enzanisa Isaya 59:15-17). Akaisa kwatiri zvokwadi yake, kururama, rugare, kutenda, ruponeso, noMweya Mutsvene (VaEfeso 6:13-17). Mwari ari mberi kwedu, isu takashonga kubva kumusoro kusvika pasi kutsoka, nenhumbi dzaakatipa, hatikundikani (VaRoma 16:20; 1 VaKorinde 15:23, 24; 2 VaTesaronika 2:8). 

`Chokwadi chokuvapo kwemasimba aya emweya – atinorwa nawo tisina simba – chinotidzidzisei pamusoro pokuti sei tichifanira kuramba takabatirira kunaJesu, uyo ari mukuru kumasimba aya uye akamakurira kare?`
