---
title:  Fedhii Gaarii fi Furmaata Waaqaa
date:   19/02/2025
---

`Efesoon 1:9–11 dubbisi. Heertuun kun waa’ee carraan namaa dursee Waaqayyoon murtaa’era jedhu ilaalchisee maal jedha? Namoonni tokko tokko fayyuuf kaan immoo akka baduuf dursee kan murteeffameedhaa?`

Asittis ta’e bakka biraatti Caaffata Qulqullaa’oo keessatti “dursanii murteessuu” (predestination) jedhamee kan hiikame jechi afaan Giriikii (prohorizo) inni jedhamu mataan isaa Waaqayyo sababa irratti hundaa’udhaan seenaa akka murteessu hin barsiisu.

Qooda kanaa, jechi afaan Giriikii kun “dursanii murteessuu” jechuudha. Dhugaadhumatti, namni tokko waan tokko dursee gama tokkootti murteessuu akka danda’u beekamaadha, ykn murtoo bilisaa kan namoota biroo tilmaama keessa galchuudhaan dursee waan tokko murteessuu akka danda’u beekamaadha. Caaffanni qulqullaa’oon Waaqayyo isa boodaa akka hojjetu barsiisa.

Asittis ta’e bakka biraatti (fakkeenyaaf, Rom. 8:29, 30), jechi “dursee murtaa’e” jedhamee hiikame, murtoo bilisaa uumamootaa ilaalchisee wanta Waaqayyo dursee beeku erga tilmaama keessa galchee booda, wanta Waaqayyo gara fuulduraatti karoorfate argisiisa. Akkasitti, walitti dhufeenya jaalala dhugaaf bilisummaa uumamaa barbaachisu kan kabaju yoo ta’e illee, Waaqayyo dhumaatti seenaa hundumaaf gara waan gaarii hawwameetti qajeelchuu danda’a.

Efesoon 1:11 Waaqayyo “wanti hundinuu akka akeeka isaatii fi akka fedha isaatiitti akka ta'u in godha” jedhee dubbata. Kana jechuun Waaqayyo wanti hundinuu akkuma inni barbaadetti akka ta’u murteessa jechuudhaa? Adda baasii dubbisi, Efesoon 1:9–11 ilaalcha kana kan mirkaneessu fakkaachuu danda’a. Haa ta’u malee, hiikni kun barreeffamoota baay’ee kanaan dura argine kan namoonni yeroo tokko tokko “fedha Waaqayyoo” morman kan faallessu agarsiisa (Luq 7:30; Luqaas 13:34, Far. 81:11–14 wajjin wal bira qabi). Macaafni Qulqulluun ofisaa hin faallessu erga ta’ee, kutaaleen kun haala wal simuun akkamitti hubatamuu danda’u?

Wanta nuti “fedhii gaarii” fi “fedhii furmaata” Waaqayyoo jechuu dandeenyu gidduu garaagarummaan akka jiru namni tokko salphaatti yoo hubate Kutaan kun hiika guutuu kan qabu ta’a. “Fedhii gaariin” Waaqayyoo wanta Waaqayyo akka ta’u filatu yeroo ta’u, kun immoo kan inni ta’u utuu namni hundi yeroo hundumaa waan Waaqayyo barbaadu hojjeteedha. “Fedhiin furmaata” Waaqayyoo gama biraatiin, fedhii Waaqayyooti, kunis murtoo bilisaa uumamootaa isa yeroo tokko tokko wanta Waaqayyo filatu irraa fagaatu dabalatee, wantoota biroo hundumaa tilmaama keessa kan galcheedha. Efesoon 1:11 “fedha furmaata” Waaqayyoo kan argisiisu fakkaata.

Waaqayyo fuuldura dursee beekuun isaa baay’ee humna guddaa waan qabuuf, filannoowwan badaa dabalatee, filannoowwan namoonni godhan hundumaa beekee illee, amma iyyuu “waan hundumaa gaarii gochuu danda’a” (Rom. 8:28). Dhugaa kanarraa jajjabina akkamii argachuu dandeessa?
