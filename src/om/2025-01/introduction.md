### Waaqa Jaalalaa fi Haqaa

Waaqayyo jaalala. 1Yohaannis 4:8 fi 16 akkas jedha. Guutummaan Macaafa Qulqulluu dhugaa kanaaf ragaa ba’a. Amantiin kiristaanaa naannoo amala jaalala Waaqayyoo wiirtuu godhata. Jaalalli eenyummaa Waaqayyoof hundeedha, wiirtuu waan nuti amannu hundaati, waan hojjennu hundumaafis hundee ta’uu qaba. Haaluma kanaan karaan jaalala itti hubannu guutummaa amantii fi shaakala keenya irratti dhiibbaa qaba. Fakkeenyaaf, namni tokko jaalalli Waaqayyoo argamuu ykn kennamuu qaba jedhee yoo amane, namni tokko cubbamaa fi kan hin malle waan ta’eef Waaqayyo akka isa hin jaallanne itti fakkaachuu danda’a. Akkasumas, namoota kaaniin walqabatee, namni tokko warri kaan jaalalli akka isaaf malu eeguu danda’a, kunis qophii balaadhaaf gargaarudha.

Karaa kanaa fi kan biroo hedduudhaan, akki nuyi jaalala Waaqayyoo itti hubannu amantii fi shaakala keenya irratti dhiibbaa guddaa qaba. Haa ta’u malee jaalalli maali? Jaalala akka ibsaniif namoota kudhan yoo gaafatte deebii adda addaa kudhan argachuu dandeessa. Kiristaanota birattillee waa’ee jaalala Waaqayyoo sheekkoo fi hubannoo dogoggoraa hedduutu jira.

Fakkeenyaaf, Kiristiyaanonni gaaffiiwwan akka: Jaalalli Waaqayyoo kennuu qofa malee gonkumaa hin fudhatuu? Jaalalli waaqaa aarsaa of gochuu qofa moo Waaqayyo namootatti gammaduu danda’a? Jaalalli Waaqayyoo miira namaa kan kakaasudhaa? Waaqayyo dhuguma namootaaf ni yaadaa? Jaalalli Waaqayyoo fudhatama dhabuu ykn mormamuu danda’aa? Waaqayyo uumamoota wajjin hariiroo jaalalaa duubatti deebi’u keessa ni seena? Aariin jaalalaa wajjin wal hin simuu? Jaalalaa fi haqni akkamitti waliin deemu? Waaqayyo jaalala yoo ta'e, maaliif hamminni baay'een isaa biyya lafaa kanarra jiraata? Ilmaan namootaa akka Waaqayyootti jaallachuu danda’uu? Yoo akkas ta’e, sun maal fakkaata?

Deebiin gaaffiiwwan kana tokko tokkoo ifaa fakkaachuu danda’a garuu yeroo baay’ee wal’aansa Kiristaanummaa jaalala waaqummaa keessatti falmisiisaa dha. Akkasumas, deebii baay’een yeroo tokko tokko ifa ta’ee fudhatamu, itti dhiyeenyaan yoo ilaalamu, wanta Caaffanni Qulqullaa’oon waa’ee jaalala Waaqayyoo barsiisu wajjin kan wal hin simne ta’ee argama.

Gaaffiiwwan kana hunda yeroo tokkotti hin ilaallu, garuu gaaffilee kunii fi kanneen biroon kurmaana kana guutuu fudhata. Akkasumas jaalalli Waaqayyoo kan nuti yaadne caalaa baay’ee akka caalu ilaalla. Jaalalli Waaqayyoo akka Caaffata Qulqullaa’oo keessatti mul’atutti yaada addunyaa keenya har’aa baay’ee keessatti “jaalala” isa darbu baay’ee caala. Torban dhufan keessatti, wantoota jaalala Waaqayyoo keessatti beekamoo fi mimmiidhagoo ta’anii Macaafa Qulqulluu keessatti mul’atan tokko tokko sirriitti ilaalla.

Akkasumas, yeroo itti fufnu, jaalalli waaqummaa fi haqni akkamitti walitti dhufeenya addaan hin cinne akka qaban ilaalla. Waaqni Macaafa Qulqulluu haqa jaallata (fakkeenyaaf Isa. 61:8 ilaali). Akkasumas, akkuma Macaafni Qulqulluun agarsiisutti, jaalalli waaqummaa fi haqni akkasitti tokko isa kaan malee jiraachuu akka hin dandeenyetti waliin deemu. Waaqayyo jaalala waan ta’eef, haqa dhabinaa fi gidiraa addunyaa kana irra jirutti baay’ee itti yaada, akkasumas warra cunqurfamanii fi dhiphatanii wajjin ofii isaa dhaaba, fedhii isaatiin dhukkubbii fi hamminatti gadduu uumama keessatti hojjete keessa seenee hundumaa caalaa ofii isaatii rakkachaa jira, kanaaf Waaqayyo ofii isaatii miidhamaa hamminaa isa guddaa ta’a.

Guutummaa Macaafa Qulqulluu keessatti Waaqayyo irra dedeebi’ee hamminaa fi gidiraadhaan gaddeera akkasumas dhukkubsateera, sababni isaa inni tokkoo tokkoo namaa kan nuti yaaduu dandeenyu caalaa waan jaallatuuf. Namni tokko gad fageenya jaalala Waaqayyoo boo’icha Kiristoos saba isaaf boo’e keessatti arguu danda’a: “Yaa Yerusaalem, yaa Yerusaalem, ishee raajota ajjeeftuu, warra sitti ergamanis dhagaadhaan rukuttuu, indaanqoon cuucota ishee qoochoo ishee jalatti akkuma walitti qabatutti, si'a meeqan ijoollee kee walitti qabuu barbaade; atii fi ijoolleen kee garuu hin barbaanne” (Mat. 23:37).

Waaqni Macaafa Qulqulluuinni inni jaalala ta’e, yeroo baay’ee guutummaa Caaffata Qulqullaa’oo keessatti jaalala tuffatamee fi jaalala badeen garaan isaa kan cabee fi kan gaddeetti fakkeeffameera. Guutummaan Seenaa Caaffata Qulqullaa’oo golee qaawwaa yunversii hundatti jaalala deebisuuf waa’ee waan Waaqayyo hojjetee fi hojjechaa jiruudha. Kun fi kanneen biroo hedduun mata duree barnoota kurmaana kanaati.
