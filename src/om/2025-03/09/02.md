---
title:  Seera Kakuu
date:   24/08/2025
---

Waaqayyo Siinaatti, seera isaa kennuudhaan karaa walitti dhufeenya isaa wajjinii akkamitti saba isaa, jireenya qulqulluu jiraachuuakkadanda’anbarsiisuufbu’uurakaa’e. Garuuqajeelfamni seerichaa jireenya guyyaa guyyaa keessatti hojii irra oolchuu waan barbaachisuuf, Waaqayyo seera dabalataa, “Seera Kakuu” jedhamu isaaniif kenne. Seerota kana ilaaluu fi sirritti hojiirra oolchuun itti gaafatamummaa abbootii murtii ture.

“Sammuu sabaa garbummaa fi amantii ormootaatiin kan jaamee fi kan gadi bu’e waan ta’eef, qajeelfama seera waaqaa kurnan guutummaatti dinqisiifachuuf qophaa’ee hin turre. Dirqamni Kurnan caalaatti akka hubatamuu fi hojiirra ooluuf, qajeelfama Abboommii Kurnanii kan ibsuu fi hojiirra kan oolchuu seeri dabalataa kennamee ture. Seerotni kun lamaan firdii jedhamu, sababni isaa isaan ogummaa fi haqummaa daangaa hin qabneen kan qophaa’anii fi abbootiin murtii isaaniin murtii kan kennan waan ta’aniif. abboommii Kurnan irraa adda waan ta’aniif, dhuunfaatti Musee isa namootatti himuu qabuutti kennaman” —Ellen G. White, Ellen G. White, Patriarchs and Prophets, p. 310.

`Seera Ba’uu 21:1–32 dubbisi. Garboota Ibrootaa, ajjeechaa fi miidhaa qaamaa ilaalchisee dambiiwwan adda taʼan akkamiitu kennaman?`

Seerri kakuu boqonnaa hedduu keessatti ibsameera (Bau. 21:1–23:19). Dambiiwwanii fi seerotni kunhundinuu badii hamminaa dhaabuu fi hawaasa sirna qabeessa ta’e ijaaruuf kan ba’an turan.

Seerri garbummaa adda waan tureef gocha gar jabinaa fi hamminaa garbummaa ammayyaa ykn bara gidduu galeessaa wajjin kan wal-dhahe ta’uu hin qabu. Dhugaa dubbachuuf, garboonni Ibrootaa, kan eegamanii fi gatii guddaa kan qaban turan. Hawaasa ammayyaa fi bara gidduu galeessaa keessatti tajaajiltoonni fi garboonni qabeenya abbaa qabeenyaa isaanii warra waan barbaadan hunda isaan irratti raawwachuu danda’anii turan. Faallaa kanaatiin, seerotni Macaafa Qulqulluu haala adda ta’een wantoota to’ata. Garbummaan waggaa jaha qofatti kan daangeffame ture (Bau. 21:1, 2; Erm. 34:8–22), waggaa torbaffaa keessattis, garboonni hundinuu gooftaa isaanii bira turuu yoo barbaadan malee bilisa ta’uu qabu turan. Gooftoonnis Sanbata boqonnaa isaaniif kennuu qabu turan (Bau. 20:9, 10) fedhii bu’uuraa isaaniis guutuu qabu turan.

`Addunyaa gara caalaa isaa keessatti gochi garbummaa hamaan dhaabbataadhaan kan haqame ta’us, karaaleen qajeelfamni isaa tokko tokko ammallee jiran maal fa’i, akkasumas, qajeelfamoota kana loluuf maal gochuu dandeenya?`
