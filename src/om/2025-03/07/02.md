---
title:  Bishaanota Hadhaawoo
date:   10/08/2025
---

Seenaa Macaafa Qulqulluu keessatti shoorri gaariis ta’e hamaan garaagaraa Namoota garagaraatiin taphatama, kanaaf gamtachuu, iddoowwan, yeroo fi yakka irratti xiyyeeffannoo guddaa kennuu qabna. Haa ta’u malee, seenaa tokko keessatti qabxiin barbaachisaan furmaataa fi barumsa argamedha. Seenaa kana keessattis kana irraa adda miti.

Akkuma kutaaleenkun agarsiisanitti Waaqayyo Rakkoo kan Furuu fi Nagaa kan Buusuudha; haa ta’u malee, hojiin Isaa amantii dhabuu namootaatiin walxaxaa ta’e. Yeroo hunda gungumuu fi abboomamuu diduu isaanii irraa kan ka’e, Ibiroonni wal xaxiinsi ciccimoon, balaan illee isaan mudate. Sababa amanuu dhabuu fi qalbii jijjiirachuu diduu isaanii irraa kan ka’e rakkina hedduu ofitti fidan.

`Ba’uu 15:22–27 dubbisi. Galaana Diimaa erga ce’anii booda, dinqii jalqabaa raawwatameef seen duubeen maal ture?`

Qormaanni amantii Israa’el inni jalqabaa barbaachisummaa bishaanii wajjin kan wal qabate yeroo ta’u, naannoo gammoojjii hamaa, ho’aa fi gogiinsaa yoo ilaalle kan nama ajaa’ibu miti. Guyyaa sadii erga imalanii booda sabni sun dhumarratti bishaan argate, garuu kan hin dhugamne ture. Maaraa jechuun “hadhaa’aa” jechuudha, bishaan sun hadhaa’aa waan tureef, Gooftaa isaanii isa isaan kunuunsu irratti amantiin Israa’el qaban dafee fajaje. Haataʼu malee, Waaqayyo gara laafina deebii kan kennu waan ta’eef dinqiin jalqabaa muka tokkoon raawwatame. Dhugaadhumattia, bishaan sana mi’aawaa fi akka dhugamu kan taasise muka utuu hin taane Gooftaa ture. Namoonni kun barumsa barbaachisoo taʼan barachuu qabu turan: (1) yeroo Gooftaa eeguuf obsa qabaachuu, akkasumas (2) Waaqayyo namoota wajjin taʼuudhaan wantoota akka raawwatu.

Haa ta’u malee, ijoolleen Israa’el waan baay’ee akka salphaatti fudhatanii dinqii gurguddaa Waaqayyo isaaniif hojjete - dinqiiwwan reefuu hinaaffaa guddaadhaan “Yaa Waaqayyo, waaqolii keessaa kan akka kee eenyu? 15:11) jedhanii galata isaaf faarfatan dafanii dagatan.

Ta’usgaruu, ergaisaangungumaniiboodaillee,“dhukkuboota” (Ba’u. 15:26) warra Gibxii dhiphisan keessaa tokkollee Israa’eloota irratti akka hin fidne Waaqayyo waadaa gale. Inni Isaaniin in eega ture. Abdii kana shaakaluu kan isaan danda’an haal-duree isaaf amanamoo ta’anii yoo turan qofa.

`Qormaataa fi wal’aansoo akkamii ofirratti fidde? Yoo isaa wajjin waliigalte Waaqayyo amma iyyuu iddoo kee buʼee akka hojjetu beekuudhaan jajjabina akkamii argachuu dandeessa?`
