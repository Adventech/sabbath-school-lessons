દુષ્ટ શક્તિઓ સામે કુસ્તી
`એફેસી.૧:૨૧, એફેસી ૩:૧૦ અને એફેસી ૬:૧૦-૨૦માં  દર્શાવવામાં આવેલી દુષ્ટ આધ્યાત્મિક શક્તિઓ માટે વિવિધ પ્રકારના શીર્ષકોની સૂચિમાં પાઉલનો હેતુ શું છે તે તમે કેવી રીતે  નક્કી કરી શકો ?`

પાઉલ  કુસ્તીબાજો વચ્ચેની સ્પર્ધા માટે ગ્રીક શબ્દ(પેલે) નો ઉપયોગ કરીને "આપણા સંઘર્ષ" (એફેસી.૬:૧૨) નું વર્ણન કરે છે.કુસ્તીને યુદ્ધ માટે ઉત્તમ તૈયારી તરીકે ગણવામાં આવતી હોવાથી,આ શસ્ત્ર-વિરુદ્ધ-શસ્ત્ર અને હાથ-થી-હાથની લડાઇનું યોગ્ય વર્ણન છે જે સૈન્ય અથડામણ થાય ત્યારે થાય છે.પાઉલ દુષ્ટ શક્તિઓ સામે વિશ્વાસીઓના નજીકના સંઘર્ષની વાસ્તવિકતા પર ભાર મૂકે છે.અહીં તેમણે તેમને આપેલા શીર્ષકો છે:

| એફેસી. ૧:૨૧ | એફેસી. ૩:૧૦ | એફેસી. ૬:૧૨ |
| --- | --- | --- |
| દરેક શાસક (અથવા દરેક નિયમ) | શાસકો | શાસકો |
| (દરેક) સત્તા | સત્તાવાળાઓ | સત્તાવાળાઓ |
| (દરેક) શક્તિ | | આ વર્તમાન અંધકાર પર દુન્યવી શક્તિઓ |
| (દરેક) આધિપત્ય | | સ્વર્ગીય સ્થાનોમાં દુષ્ટતામાં આધ્યાત્મિક શક્તિઓ |
| દરેક નામ | | |

તેના વ્યાપક વર્ણનમાં (“દરેક નામનું નામ,”એફેસી.૧:૨૧ ; “સ્વર્ગીય સ્થાનોમાં દુષ્ટતાની આધ્યાત્મિક શક્તિઓ,” એફેસી.૬:૧૨) પાઉલ ખાતરી આપે છે કે બધી દુષ્ટ અને દુન્યવી  શક્તિઓ ખ્રિસ્તના વશમાં છે. (એફેસી.૧:૨૧).જો કે,કોઈપણ યુદ્ધમાં,વિરોધી પક્ષના દળોને ઓછો અંદાજ કરવો તે ક્યારેય સારી વ્યૂહરચના નથી. પાઉલ  ચેતવણી આપે છે કે આપણે ફક્ત માનવ દુશ્મનોનો સામનો નથી કરતા,પરંતુ "સ્વર્ગીય સ્થાનોમાં દુષ્ટતાના આધ્યાત્મિક યજમાનો" (એફેસી.૬:૧૨),શેતાન એક ચાલાક સેનાપતિની આગેવાની હેઠળ છે (એફેસી.૬:૧૧).જો કે,જ્યારે આપણે આપણા શક્તિશાળી દુશ્મનો સામે સાવધ રહેવું જોઈએ,ત્યારે આપણે તેમનાથી ડરવાની જરૂર નથી.યુદ્ધમાં દેવ  આપણી સાથે હાજર છે (એફેસી.૬:૧૦) અને તેણે આપણને શ્રેષ્ઠ શસ્ત્રો,તેમનુ  પોતાનું  બખ્તર,”ઈશ્વરનું બખ્તર" પૂરું પાડ્યું છે (એફેસી.૬:૧૧;યશાયાહ ૫૯:૧૫-૧૭ ની સરખામણી કરો.).તેમણે તેમનું સત્ય,ન્યાયીપણું,શાંતિ,વિશ્વાસ અને તારણ  અને પવિત્ર આત્માને આપણા નિકાલ માટે  મૂક્યા છે (એફેસી. ૬:૧૩-૧૫). દેવ  આપણી આગળ ચાલતા હોવાથી અને તેમણે પૂરા પાડેલા બખ્તરમાં આપણે માથાથી પગ સુધીસુસજ્જ છીએ,આપણે નિષ્ફળ થઈ શકીએ નહીં (રૂમી.૧૬:૨૦ ;૧ કોરીંથી. ૧૫:૨૩,૨૪ ;૨ થેસ્સા. ૩:૮).

`આ દુન્યવી દુષ્ટ શક્તિઓની વાસ્તવિકતા શું હોવી જોઈએ - જેમની સામે આપણે, પોતે, તદ્દન લાચાર છીએ - આપણે શા માટે પ્રભુ ઈસુને પકડવા જોઈએ, જે ફક્ત આ શક્તિઓથી મહાન નથી, પરંતુ તેમને પહેલાથી જ હરાવી ચૂક્યા છે?`
