---
title:  ปล้ำสู้กับอำนาจชั่ว
date:   14/09/2023
---

`อะไรคือจุดประสงค์ของเปาโลในการให้ชื่อของอำนาจวิญญาณชั่วอย่างหลากหลายตามที่ปรากฎใน เอเฟซัส 1:21; เอเฟซัส 3:10; และเอเฟซัส 6:10-20`

เปาโลบรรยาถึง “การต่อสู้” ของเรา (เอเฟซัส 6:12) โดยใช้คำภาษากรีกสำหรับการแข่งขันมวยปล้ำ (pale) เนื่องจากมวยปล้ำถือเป็นการเตรียมตัวให้พร้อมที่ดีเยี่ยมสำหรับการต่อสู้ในสมัยนั้น นี่จึงเป็นการบรรยายที่เหมาะสมของการต่อสู้แบบใช้อาวุธประชิดตัว ซึ่งจะเกิดขึ้นเมื่อกองทัพเข้าปะทะกัน เปาโลเน้นให้เห็นถึงความจริงของการต่อสู้อย่างใกล้ชิดของผู้เชื่อกับอำนาจชั่ว

รายชื่อที่เปาโลใช้เรียกอำนาจชั่ว ดังต่อไปนี้

| เอเฟซัส 1:21 | เอเฟซัส 3:10 | เอเฟซัส 6:12 |
|---|---|---|
| ทุกภูตผีที่ครอบครอง | พวกภูตผีที่ครอบครอง | พวกภูตผีที่ครอบครอง |
| ทุกภูตผีที่มีอำนาจ | พวกภูตผีที่มีอำนาจ | พวกภูตผีที่มีอำนาจ |
| ทุกภูตผีที่มีฤทธิ์เดช | | พวกภูตผีที่ครองพิภพในยุคมืด... ในสวรรคสถาน |
| ทุกภูตผีที่ปกครอง | | |
| นามทั้งหมดที่ถูกกล่าวถึง | | |

ในคำอธิบายกว้าง ๆ ของเปาโล (“นามทั้งหมดที่ถูกกล่าวถึง” เอเฟซัส 1:21 “ต่อสู้กับพวกวิญญาณชั่วในสวรรคสถาน” เอเฟซัส 6:12) เปาโลยืนยันว่าทุกอำนาจชั่วและทุกอำนาจเหนือธรรมชาติถูกปราบปรามโดยพระคริสต์ (เอเฟซัส 1:21) ในการต่อสู้ใด ๆ ก็ตาม ไม่ควรประเมินกำลังของฝ่ายตรงข้ามต่ำเกินไป เปาโลเตือนเราว่า เราไม่เพียงแต่เผชิญหน้ากับศัตรูที่เป็นมนุษย์เท่านั้น แต่รวมถึง “ต่อสู้กับพวกวิญญาณชั่วในสวรรคสถาน” (เอเฟซัส 6:12) ซึ่งนำโดยแม่ทัพเจ้าเล่ห์ นั่นคือมาร (เอเฟซัส 6:11) อย่างไรก็ตาม ขณะที่เราต้องตื่นตัวต่อศัตรูที่ทรงพลังของเรา เราไม่จำเป็นต้องกลัวพวกมัน พระเจ้าสถิตอยู่กับเราในการสู้รบ (เอเฟซัส 6:10) และทรงจัดหาอาวุธยุทโธปกรณ์ที่ดีที่สุดแก่เรา คือยุทธภัณฑ์ของพระองค์เอง “ยุทธภัณฑ์ทั้งชุดของพระเจ้า” (เอเฟซัส 6:11; เปรียบเทียบ อิสยาห์ 59:15-17) พระองค์ทรงจัดเตรียมความจริง ความชอบธรรม สันติสุข ความเชื่อ ความรอด และพระวิญญาณบริสุทธิ์ (เอเฟซัส 6:13-17) เมื่อพระเจ้าทรงนำหน้าเราไปและเราสวมยุทธภัณฑ์ที่พระองค์ประทานให้ตั้งแต่หัวจรดเท้า เราก็จะไม่พ่ายแพ้ (โรม 16:20; 1 โครินธ์ 15:23, 24; 2 เธสะโลนิกา 2:8)

`ความจริงในเรื่องของอำนาจชั่วเหนือธรรมชาติเหล่านี้ ที่เราต้องต่อสู้กับมันอย่างไร้หนทาง สอนเราอย่างไรถึงเหตุที่เราต้องยึดจับพระเยซูไว้ ผู้ทรงไม่เพียงแต่ยิ่งใหญ่กว่าอำนาจเหล่านี้ แต่พระองค์ได้ทรงเอาชนะพวกมันแล้ว`
