---
title:  தீயவல்லமைகளோடு மற்போர்
date:   14/09/2023
---

எபே 6:12இல் பவுல், ‘நமக்குப் போராட்டம் உண்டு’ என்று சொல்கிறார். மல்யுத்த வீரர்கள் கலந்துகொள்கிற போட்டியைக் குறிக்கிற பேல் என்கிற கிரேக்க வார்த்தையைப் பயன்படுத்துகிறார். மல்யுத்த விளையாட்டுப் போருக்கான சிறந்த ஆயத்தமாகக் கருதப்பட்டது; எனவே, படைகள் கைகளோடும் ஆயுதங்களோடும் நேருக்கு நேர் மோதிக்கொள்வதைச் சுட்டிக்காட்டுகிற சரியான வார்த்தையாக உள்ளது. தீயவல்லமைகளுக்கு எதிராக நேருக்குநேர் விசுவாசிகள் போராட்டத்தில் ஈடுபட்டுள்ளதை பவுல் வலியுறுத்துகிறார்.

அந்தத் தீயவல்லமைகளின் பெயர்களைச் சொல்லுகிறார்:

| எபே 1:20 | எபே 3:10 | எபே 6:12 |
| --- | --- | --- |
| எல்லாத் துரைத்தனமும் | துரைத்தனங்கள் | துரைத்தனங்கள் |
| எல்லா அதிகாரமும் | அதிகாரங்கள் | அதிகாரங்கள் |
| எல்லா வல்லமையும் | | இப்பிரபஞ்சத்தின் அந்தகார லோகாதிபதிகள் |
| எல்லா கர்த்தத்துவமும் | | வானமண்டலங்களிலுள்ள பொல்லாத ஆவிகளின் சேனைகள் |
| பேர் பெற்றிருக்கும் எல்லா நாமமும் | | |

‘பேர்பெற்றிருக்கும் எல்லா நாமம்’, ‘வானமண்டலங்களிலுள்ள பொல்லாத ஆவிகளின் சேனைகள்’ என இயற்கைக்கு அப்பாற்பட்ட தீயவல்லமைகள் மேல் இயேசு வெற்றிபெற்றிருப்பதை பவுல் உறுதிப்படுத்துகிறார். எபே 1:20. ஆனாலும், எந்தவொரு போராக இருந்தாலும், எதிர்தரப்பு படையைக் குறைத்து மதிப்பிடுவது நல்ல உத்தியல்ல. நாம் மனித எதிரிகளை மட்டுமல்ல, தந்திரமுள்ள தளபதியான பிசாசின் தலைமையின்கீழ் இயங்குகிற ‘வானமண்டலங்களிலுள்ள பொல்லாத ஆவிகளின் சேனைகளை’ எதிர்க்கவேண்டியிருப்பதாக பவுல் சொல்கிறார். எபே 6:11,12. ஆனாலும், சக்தி வாய்ந்த நமது எதிரிகளுக்கு எதிராக எச்சரிக்கையாக இருக்கவேண்டும்; அதேசமயம், பயப்பட வேண்டியதில்லை. அந்த யுத்தத்தில் தேவன் நம்முடன் இருக்கிறார். எபே 6:10. மேலும் ஆயுதங்களிலேயே மிகச்சிறந்த ஆயுதமான, தம்முடைய சொந்த ஆயுதமாகிய ‘தேவனுடைய சர்வாயுதவர்க்கத்தை’ நமக்குக் கொடுத்திருக்கிறார். எபே 6:11; ஏசா 59:15-17. அவர் தம்முடைய சத்தியத்தையும், நீதியையும், சமாதானத்தையும், விசுவாசத்தையும், இரட்சிப்பையும், பரிசுத்த ஆவியையும் நமக்குக் கொடுத்திருக்கிறார். எபே 6:13-17. தேவன் நமக்கு முன்னே செல்ல, அவர் கொடுத்துள்ள சர்வாயுதவர்க்கத்தை தலைமுதல் கால்வரை அணிந்திருக்கும்போது, நாம் தோல்வியடைய வாய்ப்பே இல்லை. ரோமர் 16:20; 1கொரி 15:23,24; 2தெச 2:8.

`இயற்கைக்கு அப்பாற்பட்ட இந்தச் சக்திகளுக்கு எதிராக நாம் முற்றிலும் பெலனற்றவர்கள்; எனவே கர்த்தராகிய இயேசுவைப் பற்றிக்கொள்வது எவ்வளவு முக்கியம்? அவர் அந்த வல்லமைகளையும்விட பெரியவர்மட்டுமல்ல, அவற்றை அவர் ஏற்கனவே தோற்கடித்துமிருக்கிறார்.`
