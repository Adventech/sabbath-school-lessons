---
title:  Ukubambana Namandla Obubi
date:  14/09/2023
---

`Ubona kuyini injongo kaPawulu yokubhala amagama ehlukene emimoya emibi esemkhathini  kwabase-Efesu 1:21, Efesu 3:10, nesahluko 6:10-20?`

UPawulu uchaza “ngokubambana kwethu” (Efesu 6:12, esebenzisa igama lesiGriki lomncintiswano phakathi kwabantu abazama ukuwisana (pale). Njengoba lokho kwakuthathwa njengokuzilungisela impi okuhle, kuyisibonelo esifanelekile sokulwa kubhekene umuntu nomuntu, inkemba nenkemba, okwenzekayo uma kutholana phezulu amabutho. UPawulu uqhakambisa ubuqiniso bomzabalazo wabakholwayo belwa namandla obumnyama.

Nanka amagama awanika wona:

| Efesu 1:21 | Efesu 3:10 | Efesu 6:12 |
| --- | --- | --- |
| Konke ukubusa | Imibuso | Imibuso |
| (onke) amagunya | Imibuso | Imibuso |
| (onke) Amandla | | Ababusi bezwe balobu bumnyama |
| (bonke) ubukhosi | | Omoya bobubi emkhathini |
| Onke amagama aphathwayo | | |

Ngokuchaza kwakhe okubanzi (“onke amagama aphathwayo,” Efesu 1:21; “omoya bobubi emkhathini,” Efesu 6:12) uPawulu uyaqinisa ukuthi onke amandla obubi nobumnyama athotshiswe ngaphansi kukaKrestu (Efesu 1:21). Kodwa, kunoma iyiphi impi, akuwona umqondo omuhle ukubukela phansi isitha zakho. UPawulu uyaxwayisa ngokuthi asibhekene-nje nezitha zabantu, kodwa “nemikhosi yemimoya yobumnyama emkhathini” (Efesu 6:12) eholwa umkhuzi wayo oyiqili, uSathane (Efesu 6:11). Kodwa nakuba kudingeka ukuthi siqaphele izitha zethu ezinamadla, asidingi ukwesatshiswa yizo. UNkulunkulu unathi empini (Efesu 6:10) futhi usihlinzeke ngezikhali ezinhle kakhulu, ezaKhe uqobo, izikhali zikaNkulunkulu” (Efesu 6:11; qhathanisa noIsaya 59:15-17). Usinike iqiniso laKhe, ukulunga, ukuthula, ukukholwa, nosindiso, noMoya oNgcwele (Efesu 6:13-17). UNkulunkulu ehamba ngaphambi kwethu, nathi sihlonyisiwe ngezikhali azihlinzekile, kusukela ekhanda kuye ozwaneni, ngeke sehlulwe (Roma 16:20; 1 Korinte 15:23, 24; 2 Thesalonika 2:8).

`Kufanele kusifundiseni ukuba khona kwalemimoya emibi--esingenamandla kuyo--ngokuthi sidinga ukubambelela eNkosini uJesu, ongagcini-nje ngokubamkhulu kunayo, kodwa futhi oseyinqobile vele?`
