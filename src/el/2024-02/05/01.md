date:  27/04/2024

> <p>Εδαφιο Μνημης</p>
>  «Στην καρδιά μου φύλαξα τα λόγια Σου, για να μη αμαρτάνω σε Σένα.» Ψαλμός 119:11

### Για τη μελέτη αυτής της εβδομάδας, διαβάστε
Ψαλ. 119:103, 104,  Β’ Κορ. 4:1-6,  Ιωάν. 14:25, 26,  Εφεσ. 2:8, 9,  Τ.Π.Σ. κεφ. 7-11.

Το 1.500 μ.Χ. οι πρώτοι Προτεστάντες είχαν κάτι που όλοι χρειαζόμαστε. Είχαν ένα λόγο για να ζουν. Ένας διάσημος συγγραφέας, ο Philip Cushman, αναφέρεται στην ανάγκη μας για έναν λόγο για να ζούμε. Στο Βιβλίο του The Empty Self, ο Cush-man λέει ότι μερικοί άνθρωποι δεν έχουν τίποτα που να αξίζει να πεθάνουν για αυτό. Τίποτα στη ζωή δεν έχει σημασία για αυτούς. Έτσι, έχουν ελάχιστους λόγους για να ζουν.

Όμως, οι Μεταρρυθμιστές ήταν διαφορετικοί. Όπως είδαμε την προηγούμενη εβδομάδα, η Μεταρρύθμιση έγινε το 1.500 μ.Χ. στην Ευρώπη, όταν οι ηγέτες της εκκλησίας έκαναν μεγάλες αλλαγές στην Καθολική θρησκεία. Αυτοί οι ηγέτες απέρριψαν τις ψευδείς διδασκαλίες της Καθολικής εκκλησίας. Οι αλλαγές που έκαναν οδήγησαν στο ξεκίνημα των Προτεσταντικών εκκλησιών. Οι Προτεστάντες είχαν έναν σημαντικό λόγο να ζουν. Τα πιστεύω τους ήταν σημαντικά για αυτούς. Δεν μπορούσαν να τα απορρίψουν. Εάν το έκαναν, θα ήταν σαν να απέρριπταν την ταυτότητά τους.

Αυτή την εβδομάδα, θα δούμε πώς το 1.500 μ.Χ οι αλήθειες της Αγίας Γραφής άλλαξαν τη ζωή των Προτεσταντών και τους έδωσαν έναν λόγο για να ζουν. Η κατανόηση μας πάνω σε αυτές τις Βιβλικές αλήθειες μάς βοηθά να προετοιμαστούμε για τις τελευταίες δοκιμασίες των εσχάτων ημερών.
