### ΕΙΣΑΓΩΓΗ

**Μη χρονοτριβείτε, δραστηριοποιηθείτε!**

Πριν από χρόνια, ένα από τα έντυπα της εκκλησίας μας δημοσίευσε μια αλληγορία για έναν φοβερό βάλτο. Καθώς οι άνθρωποι περνούσαν από το μονοπάτι που τον διέσχιζε, συχνά έπεφταν μέσα. Οι κραυγές των ετοιμοθάνατων ακούγονταν σε όλο το γειτονικό χωριό. Ήταν τρομερό!

Οι κάτοικοι πραγματοποίησαν διάφορα συμβούλια στο χωριό. Παρουσιάστηκαν θεωρίες και έγγραφα που ανέλυαν την αιτία του προβλήματος και μερικές φορές πρότειναν κάποιες λύσεις. Ωστόσο, ποτέ δεν έγινε τίποτε άλλο εκτός από το να συνεχίσουν να συνεδριάζουν και να συζητούν. Με την πάροδο των ετών, οι συζητήσεις συνεχίστηκαν. Οι άνθρωποι έγραψαν διατριβές για το θέμα. Φέρθηκαν προσκεκλημένοι ομιλητές. Πραγματοποιήθηκαν υπαίθριες αγορές για να συγκεντρωθούν χρήματα ώστε να μπορούν να παρέχονται γεύματα σε όσους θυσίαζαν τόσες ώρες σε αυτές τις συναντήσεις. Τελικά, συγκεντρώθηκαν χρήματα για να κατασκευαστεί μια αίθουσα συνεδριάσεων με ηχομόνωση, ώστε οι κραυγές των ετοιμοθάνατων να μην διαταράσσουν τις συνεχιζόμενες συζητήσεις. Κανείς όμως δεν έκανε κάτι για να βοηθήσει εκείνους που είχαν πρόβλημα. Και κανείς δεν έκανε κάτι για να εμποδίσει άλλους ανθρώπους από το να χαθούν στον βάλτο. Έμειναν απλώς στα λόγια.

Η εκκλησία στο σύνολό της και η Σχολή Σαββάτου ειδικότερα δεν θέλουν να είναι σαν τους ανθρώπους σε εκείνο το χωριό. Μας αρέσει ο χρόνος που περνάμε μαζί για να προσευχηθούμε, να σκεφτούμε, να μοιραστούμε απόψεις και εμπειρίες και να συζητήσουμε. Όμως,  λαχταράμε να πάμε πέρα από αυτό και να κάνουμε πραγματικά κάτι μαζί για τους ανθρώπους  γύρω μας. Θέλουμε να κάνουμε τη διαφορά στις κοινότητές μας και σε όλο τον κόσμο. Θέλουμε να ολοκληρωθεί το έργο και να επιστρέψει ο Ιησούς.

Η Σχολή Σαββάτου αυτής της τριμηνίας έχει σχεδιαστεί για να εστιάσει την προσοχή μας στο να δραστηριοποιηθούμε. Θα εξετάσουμε πολλές υπέροχες ιστορίες της Αγίας Γραφής. Θα διαβάσουμε συναρπαστικές εμπειρίες και παραβολές. Θα μάθουμε για τα διαθέσιμα μέσα και τις πηγές που θα μας βοηθήσουν να προσεγγίσουμε τους συνανθρώπους μας (ειδικά εκείνους που δεν έχουν χριστιανικό υπόβαθρο). Ωστόσο, αυτά θα είναι απλώς τρόποι για να λειτουργήσουν ως παράδειγμα και να μας κινητοποιήσουν. Το πραγματικό επίκεντρο και το πραγματικό νόημα κάθε μαθήματος είναι αυτό που θα μοιραζόμαστε κάθε εβδομάδα στην περικοπή της Πέμπτης: μια πρόκληση να δραστηριοποιηθούμε.

Θα μοιραστούμε μαζί σας θεολογικές γνώσεις και θα σας δώσουμε εργαλεία και ιδέες για να εργαστείτε. Κάθε Πέμπτη θα υπάρχει το τμήμα του μαθήματος που θα θέτει μια προσεκτική εξέλιξη των προκλήσεων. Θα ξεκινήσουμε με αυτά που θεωρούνται εύκολα και καθώς η τριμηνία θα προχωράει, θα υπάρχουν διαβαθμίσεις στην πρόκληση. Ο στόχος είναι να αποδεχτούμε την πρόκληση, να προσευχηθούμε ώστε το Άγιο Πνεύμα να μας καθοδηγήσει και να εφαρμόσουμε αυτά που μάθαμε. Στη συνέχεια, την ακόλουθη εβδομάδα θα πρέπει να αφιερώσουμε λίγα λεπτά για να μοιραστούμε τις εμπειρίες μας. Αυτό δεν πρέπει να είναι μια στιγμή καυχησιολογίας αλλά μια στιγμή ανταλλαγής απόψεων, τόσο για το τι πήγε καλά όσο και για το τι δεν πήγε καλά. Καθώς μοιραζόμαστε τις εμπειρίες μας, η ομάδα θα γεννά ιδέες. Οι λίστες προσευχών θα αυξηθούν (προσωπικές και συλλογικές).

Στο τέλος, επιθυμία μας είναι αυτή η  τριμηνία να μείνει στην ιστορία όχι για τις αξιόλογες σκέψεις, τις συναρπαστικές ιστορίες ή τις βαθιές θεολογικές έννοιες. Επιθυμία μας είναι να αναπολούμε όλοι αυτή την τριμηνία ως την εποχή που το Άγιο Πνεύμα πήρε τις ταπεινές μας προσπάθειες και έκανε θαύματα για την τιμή και τη δόξα του ονόματός Του.
