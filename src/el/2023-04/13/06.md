---
title:  Η Αποστολη Ολοκληρωθηκε
date:  28/12/2023
---

`Διαβάστε Αποκ. 21:1-4, 21:22-22:5. Ποια σκηνή περιγράφουν αυτά τα εδάφια;`

Τι παράδεισος θα είναι η Νέα Γη! Ο θάνατος και η αμαρτία θα εξαφανιστούν, ο Σατανάς και η κακία θα εξαλειφθούν. Θα συναντήσουμε τον αγαπημένο μας Σωτήρα και θα επανενωθούμε με τους αγαπημένους μας. Η Νέα Γη θα κατοικηθεί με εκπροσώπους από κάθε εθνικότητα και γλώσσα.

Το Γενικό Συμβούλιο Ιεραποστολής έχει εγκρίνει μετρήσεις για την Παγκόσμια Ιεραποστολή που μπορούν να χρησιμοποιηθούν για να προσδιοριστεί αν μια ομάδα ανθρώπων έχει προσεγγιστεί ή όχι. Μια ομάδα την οποία έχουν προσεγγίσει έχει επαρκή αριθμό ατόμων και πόρους για να δώσει μαρτυρία αποτελεσματικά στην υπόλοιπη ομάδα χωρίς να χρειάζεται εξωτερική βοήθεια. Αυτό σημαίνει ότι έχουν συναθροίσεις λατρείας, Αγίες Γραφές και άλλη βιβλιογραφία στη μητρική τους γλώσσα. Επίσης, υπάρχουν ντόπιοι εκκλησιαστικοί καθοδηγητές που μπορούν να μαρτυρήσουν και σε άλλους χωρίς να εργάζονται μέσω μεταφραστή.

Μια ομάδα η οποία δεν έχει προσεγγιστεί είναι μια ομάδα που δεν έχει εγχώρια κοινότητα πιστών Αντβεντιστών με επαρκή αριθμό ατόμων και πόρων για να δώσει αποτελεσματική μαρτυρία στη δική της ομάδα χωρίς να δεχτεί βοήθεια έξω από την κουλτούρα της.

Κάθε τοπική εκκλησία και διάσκεψη πρέπει να καθορίσει τις ομάδες ανθρώπων στην κοινότητά της που χρειάζονται προσέγγιση. Τώρα είναι η ώρα να επενδύσουμε στην αποστολή του Θεού να κάνουμε μαθητές σε όλες τις ομάδες ανθρώπων, να επισπεύσουμε την επιστροφή του Σωτήρα μας και, στο τέλος, να ζήσουμε όλοι μαζί στο Νέο Ουρανό και τη Νέα Γη που μας έχει υποσχεθεί.

`Πρόκληση: Πώς επιταχύνετε την επιστροφή του Χριστού; Φυτεύετε σπόρους ελπίδας στις καρδιές εκείνων που έχουν ανάγκη να ακούσουν τα καλά νέα; “Ποτίζετε” τους νέους πιστούς βοηθώντας τους να μάθουν τι σημαίνει να ζουν μια ζωή πιστής υπακοής στον Χριστό; Προσευχηθείτε για ευκαιρίες να μεταδώσετε την υπόσχεση για τη Νέα Γη με τους ανθρώπους που βρίσκονται στην καθημερινή σας λίστα προσευχής.`

`Πρόκληση:  Κάποιοι από τους «μαθητές» σας μπορεί να είναι έτοιμοι να δεχτούν τον Χριστό. Αυτό περιλαμβάνει την ένταξή τους σε μια εκκλησία ή μια ομάδα πιστών. Βάλτε τον εαυτό σας στη θέση τους και φανταστείτε ότι πηγαίνετε στην εκκλησία σας για πρώτη φορά. Τι είδους εμπειρία θα είχαν; Πόσο προετοιμασμένη είναι η εκκλησία σας να υποδεχτεί και να μαθητεύσει καινούρια άτομα; Είστε ανοιχτοί να ξεκινήσετε νέες ομάδες πιστών και όχι απλώς να οικοδομήσετε τη δική σας υπάρχουσα εκκλησία; Δημιουργήστε μια στρατηγική για την αντιμετώπιση των αδύναμων σημείων. Μοιραστείτε τις σκέψεις σας με τους υπευθύνους της εκκλησίας σας και συνεργαστείτε μαζί τους για να εφαρμόσετε ένα πλάνο και να γίνετε μια εκκλησία που αναπτύσσει μαθητές.`
