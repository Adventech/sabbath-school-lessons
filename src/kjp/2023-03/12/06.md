---
title:  ဘၪနီၫလိၬ၁ၭလၧဆၧအၧၩ့အစီၩ့အမၩ့
date:   14/09/2023
---

`ဘၪဆဲၫ့လၧစပီလူၫအဆၧထိစိၩဘၪဆဲၫ့ (အ့ဖစၧၭ ၁း၂၁;  အ့ဖစၧၭ ၃း၁၀;  အ့ဖစၧၭ ၆း၁၀-၂၀), န၁ံၪဆီၫ့မီၪ့ထဲၩ့စပီလူၫဒုၭ၁့ၪယၫထဲၩ့ပၧၩ့ မွဲဆၧနီၪလဖၪလဲၪ.`

စပီလူၫကွ့ၭဖျီၪထၪ့ “ဆၧဒုၭအဆီၪ့” (အ့ဖစၧၭ ၆း၁၂). မွဲလၧဆၧပြၧၩ့ ဆၧဘၪဆဲၫ့လၧဆၧနီၫလိၬ၁ၭနီၪလီၫ. လၧဆၧဒုၭအဖၧၩ့ စူၪ၀ဲၩလၧခီၫစူၪ၀ဲၩဒဲ စူၪပျီၩဖျ့ၭခီၫစူၪပျီၩဖျ့ၭ ကဘၪဒုၭထဲၩ့လိၬအ၁ၭနီၪလီၫ. စပီလူၫဒုၭ၁့ၪယၫထဲၩ့ပၧၩ့  ဆၧဒုၭဒဲဆၧကၪဆၧဂဲၩအဘူၭထဲးလၧပၧၩ မွဲဆၧအၧၩ့အစီၩ့အမၩ့လဖၪနီၪလီၫ. လၧဆၧလးယီၩ ပဒၪဘၪစပီလူၫဖ့ၪလၫ့ဆၧဂဲးခိၪတၧၩ့လဖၪနီၪလီၫ.

| အ့ဖစၧၭ ၁း၂၁ | အ့ဖစၧၭ ၃း၁၀ | အ့ဖစၧၭ ၆း၁၂ |
| --- | --- | --- |
| - ကယဲဆၧအဒိၪအခိၪ | - ဆၧအဒိၪအခိၪ | ဆၧအဒိၪအခိၪ |
| - ဆၧအစီၩ့အမၩ့လဖၪ | - ဆၧအစီၩ့အမၩ့လဖၪ | ဆၧအစီၩ့အမၩ့လဖၪ |
| - ဒဲဆၧစ့ၩရၫ့ | | ဆၧအခိၪ၁ံၪ/ဆၧအစီၩ့ အမၩ့ |
| - ဒဲဆၧအကဆါ | | ဆၧခံၭအကဆါ |
| - ဒဲကယဲဆၧအမံၩ့လၧ ၦကိၭ၀့ၫလမံၩ့လါလါ | | ဆၧနၫအီၪလၧမူၭဖိးဖၫ့ |

စပီလူၫဒုၭဖျီၪထၪ့ထဲၩ့ “ကယဲဆၧအမံၩ့လၧၦကိၭ၀့ၫလမံၩ့လါလါ” (အ့ဖစၧၭ ၁း၂၁), “ဆၧနၫအီၪလၧမူၭဖိၭဖၫ့” (အ့ဖစၧၭ ၆း၁၂). ဆၧစီၩ့ဆၧမၩ့ဘၪဆဲၫ့လၧ ဆၧအၧၩ့လဖၪယီၩ အမိထၪ့ဒၩထဲၩ့ဎရံၬနီၪလီၫ (အ့ဖစၧၭ ၁း၂၁). လါနၫ့ဂ့ၩ ကယဲဆၧဒုၭလမံၩ့လါလါနီၪ စီၪဆံၭထဲၩ့ဘၪဒုၪဒၩနးဆိၭလဂ့ၩ. စပီလူၫစံၭ၀့ၫဒၪ မွဲပထၪ့ဒၩထဲၩ့ဆၧဖံၭဆၧယၫအ့ၬ, “ထၪ့ဒၩထဲၩ့ဆၧအစီၩ့အမၩ့ဒဲထၪ့ဒၩအယီၩ လကၠၪအဆၧခံၭအကဆါ, ဆၧနၫအီၪလၧမူၭဖိၭဖၫ့” (အဖ့စၧၭ ၆း၁၂) နီၪလီၫ. ဆၧနီၪအကၠီၩ့နမံၫခံၫ့မံၫယူၭလန့ဘၪနီၪလီၫ. ၦဒုၪဒၩယီၩအစီၩအမၩ့ဎိၩဒိၪအၪအၪ လ့ၬ. လါနၫ့ဂ့ၩယွၩဖ့ၪလၫ့ထဲၩ့ပၧၩ “ယွၩအစူၪ၀ဲၩ (အ့ဖစၧၭ ၆း၁၁) နီၪလီၫ. ထိၪယိထဲၩ့လၧ (အဲၫစ့ယၫ ၅၉း၁၅-၁၇). အ၀့ၫဖ့ၪလၫ့ထဲၩ့ဆၧမွဲဆၧတီၩ, ဆၧတီၩဆၧလီၩ့, ဆၧဘၪလိၬအိၭလိၬ, ဆၧနဲၩ, ဆၧထိးထဲၩ့တၩ့ထဲၩ့, ဒဲ၁ၭမုဆံၫ့ဎံၪနီၪလီၫ. (အ့ဖစၧၭ ၆း၁၃-၁၇). ကဆါယွၩလ့ၩလၧပမ့ၬယၫတၭ ၦကိၭဂၩအ့ၪမွဲပ၁ိၭထၪ့ယွၩအစူ၀ဲၩထီးလူၬလဖၪနီၪ ဒုၪဒၩမၩနၧၩ့ထဲၩ့ပၧၩမွဲကန့၁့ၪအ့ၬ.” (ရိမၫ ၁၆း၂၀;  ၁ကီရံၫ့ ၁၅း၂၃, ၂၄; ၂၁့ၭစလနဲၫကၫ ၂း၈).

`ပဘၪဒၪဆၧၫ့ယၫထဲၩ့ဆၧစီၩ့ဆၧမၩ့လဖၪယီၩလါနၫ့ဂ့ၩ ယွၩမၩနၧၩ့ခွံၬ အ၀့ၫ၁ံၪ ခီၫလၧဖျိၩ့အိၩအကၠီၩ့ ၦဘၪ၁ုၪ့မဲၪ၁ၭဖျဲၩ့ဆၧအလးအီၪနးမံၩ့အ့ၬ.`
