### တၢ်မၤလိတနွံအံၤအဂီၢ်ဖး
၂မိၤၡွ ၂၅း၈, ၉, ၄၀း ဧ့ၤဘြံၤ ၈း၁-၆; ၃မိၤၡွ ၁၆း၂၁, ၂၉-၃၄; ၃မိၤၡွ ၂၃း၂၆-၃၂; ဧ့ၤဘြံၤ ၉း၂၃-၂၈; ဒၤနံးယ့လး ၇း၉, ၁၀.

> <p>တၢ်ကလုၢ်ထူ</p>
> "ဒီးတၢ်လၢပကတိၤတၢ်န့ၣ်အမိၢ်ၦၢ်မ့ၢ်အအံၤ, ပအိၣ်ဒီးၦၤလုၢ်တၢ် အခိၣ်ထဲထဲန့ၣ်, လၢအဆ့ၣ်နီၤဝဲလၢတၢ်ကဟုကညီၢ်အလီၢ်ပစိၥ်အစုထွဲလၢ မူခိၣ်လီၤ. အဝဲဒၣ်န့ၣ် မ့ၢ်တၢ်အလီၢ်စီဆှံ, ဒီးဒဲနီၢ်နီၢ်လၢကစၢ်သူၣ်ဝဲ, ဒီးတမ့ၢ်ဘၣ်ၦၤကညီဘၣ်န့ၣ်, အခ့အၦၤလီၤ" (ဧ့ၤဘြံၤ ၈း၁, ၂).

တၢ်သူၣ်ဟးဂူၥ်သးဂီၤဖဲလါ October ၂၂, ၁၈၄၄ ၀ံၤအလီၢ်ခံ တယံၥ်ဘၣ်န့ၣ်, ၦၤပိၥ် Miller အခံသ့ၣ်တဖၣ် နၢ်ပၢၢ်ဝဲလၢ၀ံအကလုၢ်ကထါ အသီ ၂၃၀၀ အံၤန့ၣ် တဘၣ်ဃးဒီးကစၢ်ဟဲက့ၤလီၤခံဘျီတဘျီအဂ့ၢ်ဘၣ်, မ့မ့ၢ်တခီဘၣ်ဃးကစၢ်ခ၇ံၥ်အတၢ်မၤလၢမူခိၣ်တၢ်လီၢ်စီဆှံအပူၤန့ၣ်လီၤ. တၢ်မၤကဆှီကွံၥ်တၢ်လီၢ်စီဆှံလၢမူခိၣ်ဘီမုၢ်အပူၤအံၤ မ့ၢ်တၢ်မၤလၢၦဲၤ တၢ်သ့ကဆှီဟီၣ်ခိၣ်အတၢ်လီၢ်စီဆှံအံၤန့ၣ်လီၤ. ဒ်သိးကနၢ်ပၢၢ်တၢ်မ့ၢ်တၢ်တီ အ၇့ဒိၣ်မးအံၤ ကဂ့ၤဒိၣ်ထီၣ်အဂီၢ်, ကွၢ်တၢ်သဃဲၤထီၣ်လၢ ဒၤနံးယ့လး ၇ ဒီး ၈ အပူၤတက့ၢ်.

| ဒၤနံးယ့လး ၇ | ဒၤနံးယ့လး ၈ |
| --- | --- |
| Babylon (ဘၤဘူၤလိၣ်) | |
| မၤဒဲ-ပၤ၇ၣ် | Media-Persia (မၤဒဲ-ပၤ၇ၣ်) |
| Greece (ယၤ၀ၣ်) | Greece (ယၤ၀ၣ်) |
| Rome (၇ိမ့ၤ) |. Rome (၇ိမ့ၤ) |
| တၢ်စံၣ်ညီၣ်လၢမူခိၣ် | တၢ်မၤကဆှီတၢ်အလီၢ်စီဆှံ |

တၢ်သဃဲၤထီၣ်သ့ၣ်တဖၣ်အံၤ ဒုးနဲၣ်ဖျါတၢ်မၤကဆှီတၢ်လီၢ်စီဆှံလၢ အမ့ၢ်တချုးကစၢ်ခ၇ံၥ်ဟဲက့ၤဒ်ဘၣ်အကတီၢ်န့ၣ်လီၤ. တနွံအံၤပကလူၤဃု သ့ၣ်ညါကစၢ်ခ၇ံၥ်အတၢ်မၤလၢမူခိၣ်ဘီမုၢ်အပူၤန့ၣ်လီၤ.
